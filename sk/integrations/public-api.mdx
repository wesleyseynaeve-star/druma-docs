## Prehľad

Druma poskytuje REST API, ktoré vám umožňuje programovo interagovať s vašimi dátami Druma. Použite ho pre:
- Integráciu Druma s vašim ERP alebo WMS systémom
- Automatické vytváranie zákaziek z externých zdrojov
- Načítanie stavu zákazky a polohy pre interné dashboardy

## Základná URL

```
https://api.druma.io/v1/
```

## Autentifikácia

Všetky API požiadavky musia obsahovať API kľúč v hlavičke:

```
Authorization: Bearer YOUR_API_KEY
```

Generujte API kľúče v **Nastavenia → Integrácie → Verejné API → Generovať kľúč**.

## Hlavné endpointy

| Endpoint | Metóda | Popis |
|----------|--------|-------|
| `/orders` | GET | Zoznam zákaziek s filtrovaním |
| `/orders` | POST | Vytvorenie novej zákazky |
| `/orders/{id}` | GET | Získanie detailov jednej zákazky |
| `/orders/{id}/status` | GET | Aktuálny stav zákazky |
| `/clients` | GET | Zoznam klientov |
| `/fleet/vehicles` | GET | Zoznam vozidiel |
| `/fleet/drivers` | GET | Zoznam vodičov |

## Obmedzenie rýchlosti

- **100 požiadaviek za minútu** pre štandardné API kľúče
- Vráti HTTP 429 keď je limit prekročený
- Záhlavie `X-RateLimit-Remaining` ukazuje zostatok
