## Přehled

TransFollow je certifikovaná platforma eCMR, kterou Druma používá pro vydávání, podepisování a archivaci elektronických nákladních listů (eCMR). Bez této integrace není eCMR dostupné.

## Požadavky

- Aktivní účet TransFollow (zaregistrujte se na [transfollow.org](https://transfollow.org))
- API klíč z panelu TransFollow

## Konfigurace

<Steps>
  <Step title="Přejděte do Nastavení → Integrace">
    Otevřete kartu **TransFollow**.
  </Step>
  <Step title="Vložte API klíč">
    Zkopírujte API klíč z vašeho účtu TransFollow a vložte ho.
  </Step>
  <Step title="Volitelně povolte zkušební režim">
    Zkušební režim umožňuje testování bez vydávání právně platných dokumentů.
  </Step>
  <Step title="Otestujte a uložte">
    Klikněte na **Otestovat připojení**, pak **Uložit**.
  </Step>
</Steps>

<Warning>
Dokumenty eCMR ve zkušebním režimu nejsou právně platné. Vypněte zkušební režim před ostrým provozem.
</Warning>
