Druma poskytuje API endpointy pro klienty, kteri chteji integrovat data zasilek do vlastnich systemu.

## Autentizace

Klientsky API pristup vyzaduje API klic specificky pro klienta. Vygenerujte ho v profilu klienta v sekci **API pristup**.

## Dostupne endpointy

- GET /client/shipments - seznam zasilek klienta
- GET /client/shipments/{id} - podrobnosti zasilky
- GET /client/documents/{id} - stazeni dokladu
- GET /client/invoices - seznam faktur
