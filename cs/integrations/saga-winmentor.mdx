## Přehled

Druma generuje exportní soubory kompatibilní se SAGA Classic a WinMENTOR — dvěma nejrozšířenějšími rumunsko-orientovanými účetními programy. Export zahrnuje faktury prodeje, faktury nákupu, platby a záznamy zákazníků.

## SAGA Classic a WinMENTOR Classic

Není potřeba API klíč. Přejděte do **Reporty → Export účetnictví**, vyberte rozsah dat a formát, stáhněte soubor a importujte ho do vašeho účetního softwaru.

## WinMENTOR Enterprise (REST API)

WinMENTOR Enterprise podporuje přímou API integraci.

<Steps>
  <Step title="Otevřete kartu WinMENTOR Enterprise">
    V **Nastavení → Integrace** klikněte na **WinMENTOR Enterprise**.
  </Step>
  <Step title="Zadejte REST endpoint a přihlašovací údaje">
    Zadejte URL API endpointu a přihlašovací údaje od správce WinMENTOR.
  </Step>
  <Step title="Otestujte a uložte">
    Klikněte na **Otestovat připojení**, pak **Uložit**.
  </Step>
</Steps>
