## Přehled

SmartBill je rumunská cloudová fakturační platforma. S aktivní integrací Druma automaticky odesílá potvrzené faktury do SmartBill — bez ručního opakovaného zadávání.

## Konfigurace

<Steps>
  <Step title="Získejte API klíč SmartBill">
    V účtu SmartBill přejděte do **Účet → API** a zkopírujte API klíč.
  </Step>
  <Step title="Otevřete kartu SmartBill v Druma">
    Přejděte do **Nastavení → Integrace → SmartBill**.
  </Step>
  <Step title="Zadejte API klíč a CIF">
    Vložte API klíč a zadejte CIF vaší společnosti.
  </Step>
  <Step title="Otestujte a uložte">
    Klikněte na **Otestovat připojení**, pak **Uložit**.
  </Step>
</Steps>

## Jak to funguje

Jakmile je faktura v Druma označena jako potvrzená, je automaticky odeslána do SmartBill. Na stránce faktury se zobrazí stav SmartBill (Odesláno, Odmítnuto, Chyba).

<Warning>
Pokud vyprší předplatné SmartBill nebo je API klíč regenerován, integrace tiše přestane fungovat. Pravidelně kontrolujte stav v Nastavení → Integrace.
</Warning>
