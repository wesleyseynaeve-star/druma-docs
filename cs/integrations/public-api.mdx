## Přehled

Druma poskytuje REST API pro integraci s vlastními systémy, ERP platformami nebo vlastními dashboardy. API je dostupné pro společnosti na placených plánech.

## Autentizace

Druma API používá klíče API pro autentizaci. Vygenerujte klíč API v **Nastavení → Integrace → API**.

Zahrňte klíč do každého požadavku v hlavičce:
```
Authorization: Bearer <vas-api-klic>
```

## Klíčové endpointy

| Endpoint | Popis |
|---|---|
| `GET /orders` | Seznam objednávek s filtrováním |
| `GET /orders/{id}` | Podrobnosti objednávky |
| `POST /orders` | Vytvoření nové objednávky |
| `GET /vehicles` | Seznam vozidel |
| `GET /drivers` | Seznam řidičů |
| `GET /invoices` | Seznam faktur |

## Webhooky

Nakonfigurujte webhooky v **Nastavení → Integrace → Webhooky** pro přijímání real-time notifikací při:
- Změně stavu objednávky
- Vytvoření faktury
- Doručení zásilky

## Limity sazeb

100 požadavků za minutu na klíč API. Překročení limitu vrátí HTTP 429.
