Veřejné API Druma poskytuje **přístup pouze pro čtení** REST k vašim objednávkám — užitečné pro zákaznický samoobslužný portál, BI/reportovací nástroj nebo interní nástroje vytvořené vaším týmem.

<Warning>
Veřejné API je funkce skrytá za feature flagem a ve výchozím nastavení není povolena. Kontaktujte **support@druma.io** pro požádání o přístup — API klíče v současnosti vydává přímo Druma; v aplikaci neexistuje samoobslužná obrazovka pro generování klíčů.
</Warning>

## Autentizace

Každý požadavek API musí obsahovat váš API klíč v hlavičce `X-API-Key`:

```
X-API-Key: VAS_API_KLIC
```

Požadavky bez platného klíče vrací odpověď `401 Unauthorized`. Autentizace bearer tokenem (`Authorization: Bearer ...`) **není** podporována a také vrací 401.

## Dostupné endpointy

API je pouze pro čtení — přijímají se pouze požadavky `GET`. Všechny odpovědi jsou JSON.

| Metoda | Endpoint | Popis |
|---|---|---|
| `GET` | `/orders` | Seznam objednávek (stránkovaný, volitelný filtr `status`) |
| `GET` | `/orders/{id}` | Podrobnosti objednávky |
| `GET` | `/orders/{id}/location` | Aktuální pozice vozidla a poslední ETA |
| `GET` | `/orders/{id}/documents` | Seznam dokumentů s časově omezenými podepsanými URL pro stažení |

Jakýkoli jiný zdroj, nebo jakákoli metoda kromě GET, vrací chybu `404`/`405`.

## Limity požadavků

- **100 požadavků za minutu** na API klíč
- Pokud limit překročíte, obdržíte odpověď `429 Too Many Requests` s hlavičkou `Retry-After` udávající, kolik sekund počkat
- Žádné jiné hlavičky limitu sazby se nevrací

Pro vyšší limity kontaktujte support@druma.io se svým případem použití.
