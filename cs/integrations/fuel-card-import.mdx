## Přehled

Druma se nepřipojuje přímo k API poskytovatelů palivových karet. Místo toho stáhnete CSV export z webového portálu vašeho poskytovatele a nahrajete ho do Druma. Transakce jsou automaticky spárovány s vozidly podle registrační značky.

## Podporovaní poskytovatelé

- DKV
- UTA
- AS24
- Shell Card
- SmartDiesel

## Import transakcí

<Steps>
  <Step title="Export z portálu poskytovatele">
    Přihlaste se do webového portálu vašeho poskytovatele palivové karty a exportujte transakce jako CSV.
  </Step>
  <Step title="Přejděte do Nastavení → Integrace → Import palivových karet">
    Otevřete kartu Import palivových karet.
  </Step>
  <Step title="Vyberte poskytovatele">
    Vyberte vašeho poskytovatele z rozbalovací nabídky.
  </Step>
  <Step title="Nahrajte CSV">
    Klikněte na **Nahrát CSV** a importujte soubor.
  </Step>
</Steps>

## Nespárované transakce

Pokud registrační značka transakce neodpovídá žádnému vozidlu v parku, transakce je označena jako **Nespárovaná**. Pravidelně kontrolujte seznam nespárovaných — odhaluje podvody nebo nesprávné zadání značky.
