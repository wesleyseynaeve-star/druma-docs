Palivo je typicky největší proměnlivý náklad v silniční přepravě. Druma umožňuje importovat data transakcí od hlavních poskytovatelů palivových karet a propojit každé natankování se správným vozidlem, takže vaše nákladové reporty odrážejí skutečné výdaje za palivo místo odhadů.

## Podporovaní poskytovatelé

Druma automaticky rozpozná poskytovatele podle záhlaví sloupců a názvu souboru. Podporovaní poskytovatelé:

- **DKV**
- **UTA**
- **AS24**
- **Shell**
- **TotalEnergies**
- **BP**
- **Circle K**
- **Eurowag**
- **WEG**
- **MOL**
- **E100**

Přijímá se i jakýkoli jiný soubor CSV nebo XLSX se standardními názvy sloupců (datum, částka, litry, karta/značka), a to i bez rozpoznaného poskytovatele. Pokud používáte více karet napříč flotilou, můžete importovat od každého poskytovatele samostatně.

## Jak importovat

<Steps>
  <Step title="Stáhněte soubor od poskytovatele">
    Přihlaste se do online portálu vašeho poskytovatele palivové karty a exportujte historii transakcí jako CSV nebo XLSX. Portál každého poskytovatele je jiný, ale tato možnost bývá obvykle pod Reporty nebo Výpisy.
  </Step>
  <Step title="Otevřete import paliva Druma">
    V Druma přejděte do **Vozový park → Palivo** a klikněte na **Importovat palivo**.
  </Step>
  <Step title="Nahrajte soubor">
    Přetáhněte nebo vyberte soubor CSV/XLSX (max 5 MB). Druma automaticky rozpozná poskytovatele ze souboru a zobrazí náhled zpracovaných řádků — včetně jakýchkoli řádků mýtného, které najde smíchané ve stejném výpisu.
  </Step>
  <Step title="Zkontrolujte a potvrďte">
    Zkontrolujte náhled — řádky bez odpovídajícího vozidla jsou označeny a vyloučeny z importu. Klikněte na **Importovat N řádků** (zobrazuje skutečný počet řádků) pro potvrzení.
  </Step>
</Steps>

## Co se importuje

Pro každou transakci Druma extrahuje:

| Pole | Zdroj |
|---|---|
| Datum (bez časové složky) | Sloupec data transakce ze souboru |
| Vozidlo | Spárováno podle čísla palivové karty, poté podle registrační značky |
| Název čerpací stanice | Pole umístění ze souboru |
| Natankované litry | Pole množství ze souboru |
| Částka | Pole hodnoty ze souboru — předpokládá se EUR, pokud není přítomen sloupec měny |
| ID transakce | Jedinečná reference transakce poskytovatele, použitá pro detekci duplicit |

Ruční jednotlivé zadání natankování (přes **Přidat natankování**) přijímá pouze částku v EUR — na tomto formuláři není žádný výběr měny. Částky RON se automaticky převádějí pouze přes samostatnou cestu automatického příjmu e-mailem/vhazovací složkou popsanou níže.

## Párování transakcí s vozidly

Druma páruje každou transakci s vozidlem, nejprve podle **čísla palivové karty**, poté podle **registrační značky**. Číslo karty i značka jsou před porovnáním normalizovány — převedeny na velká písmena, se všemi mezerami, pomlčkami a jinými nealfanumerickými znaky odstraněnými — takže „B 123 ABC" a „B123ABC" odpovídají stejnému vozidlu.

Pokud transakce stále neodpovídá žádnému vozidlu, je v náhledu označena jako **nespárovaná** a vyloučena z importu. Časté důvody:

- Vozidlo bylo nedávno přidáno a ještě není v Druma
- Ani číslo karty, ani značka na transakci neodpovídají žádnému záznamu flotily
- Řidič použil svou kartu v pronajatém nebo subdodavatelském vozidle

## Detekce duplicit

Pokud omylem importujete stejný soubor dvakrát, Druma detekuje duplicity pomocí deduplikačního klíče sestaveného z data transakce, částky, litrů, karty, značky a ID transakce poskytovatele — duplicity se automaticky přeskočí na úrovni databáze. Zpráva o úspěchu hlásí pouze počet importovaných natankování (a řádků mýtného); nehlásí samostatný počet přeskočených duplicit.

<Note>
Vždy importujte kompletní výpis místo částečného exportu. Částečné importy mohou způsobit mezery v datech nákladů na palivo, které ovlivňují výpočty ziskovosti tras.
</Note>

## Automatický import e-mailem

Kromě ručního nahrávání může Druma automaticky přijímat výpisy paliva z příchozí e-mailové adresy (se záložním naplánovaným průchodem), přičemž znovu použije stejný analyzátor jako ruční importní okno. Pro nastavení tohoto pro váš účet kontaktujte podporu Druma.

## Kde se importovaná data o palivu zobrazují

Po importu se transakce zobrazují v:

- **Vozový park → Palivo** — úplná historie transakcí napříč vaší flotilou, filtrovatelná a prohledávatelná podle značky, poskytovatele a umístění
- **Report ziskovosti tras** — importované náklady na palivo nahrazují odhadovaný výpočet paliva u objednávek, kde jsou k dispozici skutečná data o palivu

## Proč import CSV místo živého API?

Poskytovatelé palivových karet napříč Evropou mají nekonzistentní a často omezený přístup k API. Někteří poskytují API pouze podnikovým zákazníkům, jiní mají portály specifické pro danou zemi bez přístupu k API a formáty API se výrazně liší. Export CSV je univerzálně dostupný u všech poskytovatelů a všech typů účtů, což z něj dělá nejspolehlivější přístup pro rumunského operátora se smíšenou flotilou (ADR-016).
