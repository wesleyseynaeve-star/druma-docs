## Przegląd

Druma rejestruje wartość licznika na początku i końcu każdej podróży, a następnie oblicza, ile kilometrów ciężarówka faktycznie przejechała — osobno od odległości trasy planowanej przez HERE. Pozwala to wykrywać odchylenia trasy, identyfikować kierowców, którzy konsekwentnie przekraczają planowane dystanse, i uzgadniać koszty paliwa z rzeczywistymi km.

---

## Jak to działa

Gdy zlecenie przechodzi do statusu **W trasie**, Druma rejestruje `odometer_start_km` z telematyki ciężarówki. Gdy zlecenie przechodzi do statusu **Dostarczone**, rejestruje `odometer_end_km`.

```
actual_distance_km = odometer_end_km − odometer_start_km
```

Obie wartości i obliczone `actual_distance_km` są pokazane w **panelu szczegółów zlecenia** w sekcji trasy, obok planowanej odległości HERE.

---

## Obsługiwani dostawcy telematyki

Dane licznika pochodzą wyłącznie z integracji telematycznych. PWA kierowcy nie rejestruje odczytów licznika ręcznie.

| Dostawca | Uwagi |
|---|---|
| **Geotab** | Pełny odczyt licznika przez MyGeotab SDK |
| **Webfleet** | Pole licznika API WEBFLEET.connect |
| **VDO** | Telematyka floty przez API dostawcy |
| **Frotcom** | Przez API dostawcy (beta) |
| **Webeye / Eurowag** | Przez API dostawcy (beta) |
| **rFMS (Scania, Volvo, DAF, MAN, Mercedes-Benz, IVECO, Renault)** | Całkowity dystans pojazdu przez adapter standardu rFMS |

Wszyscy dostawcy przesyłają dane przez funkcję Edge `position-normaliser` Druma, która zapisuje jeden wiersz per ciężarówka przy każdym piknięciu pozycji (wzorzec nadpisywania jednego wiersza RODO).

---

## Jeśli pole jest puste

`actual_distance_km` jest puste gdy:
- Brak skonfigurowanej integracji telematycznej
- Telematyka nie zwróciła wartości licznika przy wyjeździe lub dostawie
- Ciężarówka została zmieniona w połowie zlecenia przez **Zmiana ciężarówki**

Aby połączyć dostawcę telematyki, przejdź do **Ustawienia — Integracje** i skonfiguruj dane uwierzytelniające dostawcy.

---

## Planowana vs rzeczywista odległość

Panel szczegółów zlecenia pokazuje obie wartości obok siebie:
- **Planowana odległość** — obliczona przez HERE Routing w chwili tworzenia zlecenia
- **Rzeczywista odległość** — obliczona z delty licznika telematyki

Znaczna różnica (ponad ~15%) może wskazywać na objazd, błąd routingu lub nieodnotowany przystanek.
