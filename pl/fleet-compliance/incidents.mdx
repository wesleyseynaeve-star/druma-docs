## Przegląd

Karta Zdarzenia w module Floty to ustrukturyzowany dziennik wszelkich wypadków, zdarzeń szkodowych lub incydentów bezpieczeństwa z udziałem Twoich pojazdów.

**Dostęp:**
- Na pojazd: Flota → Ciężarówki → otwórz pojazd → karta **Zdarzenia**
- Dla całej floty: Flota → **Zdarzenia**

---

## Tworzenie zdarzenia

<Steps>
  <Step title="Otwórz widok zdarzeń">
    Otwórz konkretny pojazd i kliknij kartę **Zdarzenia**, lub przejdź do **Flota → Zdarzenia**.
  </Step>
  <Step title="Kliknij Nowe zdarzenie">
    Kliknij **+ Nowe zdarzenie**. Otworzy się formularz.
  </Step>
  <Step title="Wypełnij szczegóły">
    Wypełnij pola formularza (patrz tabela poniżej).
  </Step>
  <Step title="Dołącz dokumenty i zdjęcia">
    Dołącz zdjęcia szkód, raport policji PDF, korespondencję ubezpieczeniową lub oświadczenie kierowcy.
  </Step>
  <Step title="Zapisz">
    Kliknij **Zapisz**.
  </Step>
</Steps>

### Pola formularza

| Pole | Opis |
|---|---|
| **Data i godzina** | Kiedy zdarzenie miało miejsce |
| **Lokalizacja** | Adres lub współrzędne GPS |
| **Kierowca** | Który kierowca prowadził pojazd |
| **Typ zdarzenia** | Kolizja / Szkoda / Prawie-wypadek / Kradzież / Wandalizm / Inne |
| **Opis** | Relacja w dowolnym tekście |
| **Strona trzecia** | Włącz jeśli inny pojazd był zaangażowany |
| **Numer raportu policji** | Referencja jeśli złożono raport |
| **Ranni** | Włącz i opisz jeśli ktoś został ranny |

## Śledzenie roszczeń ubezpieczeniowych

| Pole | Opis |
|---|---|
| **Numer referencyjny roszczenia** | Referencja Twojego ubezpieczyciela |
| **Status roszczenia** | Otwarte / W trakcie weryfikacji / Rozliczone / Odrzucone |
| **Kwota rozliczenia (€)** | Wprowadź przy rozliczeniu |
