Użyj importu masowego, aby dodać wszystkie pojazdy naraz — idealne podczas rozpoczynania lub migracji z innego systemu.

## Pobierz szablon

<Steps>
  <Step title="Otwórz stronę importu">
    Przejdź do **Flota → Pojazdy → Importuj** i kliknij **Pobierz szablon**.
  </Step>
  <Step title="Wypełnij CSV">
    Otwórz plik w Excelu lub Google Sheets. Wypełnij jeden pojazd per wiersz:

    | Kolumna | Wymagana | Przykład |
    |---------|----------|---------|
    | plate | Tak | B 123 ABC |
    | type | Tak | tractor, rigid, van |
    | trailer_type | Nie | curtainsider, reefer, flatbed |
    | euro_standard | Tak | Euro 3 – Euro 6 |
    | height_m | Nie | 4.0 |
    | weight_tonnes | Nie | 24 |
    | axle_count | Nie | 3 |
    | payload_kg | Nie | 24000 |
    | daily_cost_eur | Nie | 120 |
    | adr_expiry | Nie | 2025-12-31 |
    | active | Tak | true lub false |
  </Step>
  <Step title="Prześlij plik">
    Zapisz jako CSV. Z powrotem w Druma kliknij **Prześlij CSV** i wybierz plik.
  </Step>
  <Step title="Przejrzyj zmapowane rekordy">
    Druma pokazuje tabelę podglądu ze wszystkimi wykrytymi wierszami. Wiersze z błędami są podświetlone na czerwono — popraw je w CSV i prześlij ponownie lub edytuj inline.
  </Step>
  <Step title="Potwierdź import">
    Kliknij **Importuj [X] pojazdy**. Pojazdy są tworzone natychmiast i pojawiają się na liście floty.
  </Step>
</Steps>

## Edycja cyklu

Ten sam szablon działa do aktualizacji. Przejdź do **Flota → Pojazdy → Eksportuj CSV**, edytuj wyeksportowany plik i ponownie zaimportuj. Druma dopasowuje po numerze rejestracyjnym i aktualizuje istniejące rekordy zamiast tworzyć duplikaty.

<Note>
Numery rejestracyjne muszą być unikalne. Jeśli ponownie zaimportujesz numer rejestracyjny, który już istnieje, Druma zaktualizuje ten pojazd zamiast tworzyć drugi wpis.
</Note>

<Warning>
Ustawienie `active = false` oznacza pojazd jako nieaktywny — zniknie z rozwijanych list przydziału. Dokładnie sprawdź tę kolumnę przed importem.
</Warning>
