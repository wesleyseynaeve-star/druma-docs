Dodaj cały zespół kierowców naraz przy użyciu importu CSV — niezbędne podczas wdrażania floty 10 lub więcej kierowców.

## Pobierz szablon

<Steps>
  <Step title="Otwórz stronę importu">
    Przejdź do **Flota → Kierowcy → Importuj** i kliknij **Pobierz szablon**.
  </Step>
  <Step title="Wypełnij CSV">
    Jeden kierowca per wiersz:

    | Kolumna | Wymagana | Przykład |
    |---------|----------|---------|
    | full_name | Tak | Gheorghe Ionescu |
    | phone | Tak | +40721000000 |
    | email | Nie | ghe@example.com |
    | license_number | Nie | B-123456 |
    | license_category | Nie | CE |
    | license_expiry | Nie | 2026-08-15 |
    | cpc_expiry | Nie | 2025-09-30 |
    | tachograph_card | Nie | RO123456780 |
    | adr_expiry | Nie | 2025-12-31 |
    | country | Nie | RO |
    | active | Tak | true lub false |
  </Step>
  <Step title="Prześlij i przejrzyj">
    Prześlij CSV. Sprawdź podgląd pod kątem czerwonych wierszy (zazwyczaj brakujący numer telefonu lub nieprawidłowy format daty). Daty muszą być w formacie `YYYY-MM-DD`.
  </Step>
  <Step title="Potwierdź import">
    Kliknij **Importuj [X] kierowców**. Kierowcy natychmiast pojawiają się na liście.
  </Step>
</Steps>

## Numer telefonu jest kluczowy

Numer telefonu to sposób, w jaki kierowcy logują się do PWA. Musi być w formacie międzynarodowym (`+40...` dla Rumunii). Bez prawidłowego numeru telefonu kierowca nie może uzyskać dostępu do aplikacji.

<Note>
Po imporcie udostępnij link do PWA każdemu kierowcy. Przejdź do **Flota → Kierowcy → wybierz kierowcę → Kopiuj link PWA** i wyślij przez WhatsApp lub SMS.
</Note>

## Alerty wygaśnięcia

Jeśli wypełnisz daty wygaśnięcia prawa jazdy, CPC lub ADR podczas importu, Druma natychmiast zacznie śledzić te daty i alertować Cię 30 i 60 dni przed wygaśnięciem.
