## Qu'est-ce qu'un contrat de capacité ?

Un contrat de capacité est un engagement formel à dédier un camion spécifique à un client pour une période définie. Le client paie un forfait mensuel fixe, un tarif au km, ou une combinaison des deux — indépendamment du fait que chaque jour soit pleinement utilisé ou non. Courant dans les relations chargeur à long terme où le client veut une capacité garantie.

Contrats de capacité se trouve sous la section **Finance**. Accessible aux rôles admin, company_admin, et fleet_manager.

## Champs du contrat

Lors de la création ou modification d'un contrat, vous renseignez :

| Champ | Description |
|---|---|
| Camion | Le camion dédié |
| Client | Le client recevant la capacité dédiée |
| Tarif par km | Revenu gagné par km chargé (EUR ou RON) |
| Forfait mensuel fixe | Montant mensuel fixe, indépendant de l'utilisation |
| Devise | EUR ou RON |
| Date de début | Quand le contrat devient actif |
| Date de fin | Quand le contrat expire (laissez vide pour une durée indéterminée) |
| Statut | Actif, Suspendu, ou Terminé |
| Notes | Notes internes — non visibles par le client |

---

## Cycle de vie du statut

```
Actif → Suspendu → Actif   (suspension temporaire, ex. véhicule hors service)
Actif → Terminé            (terme du contrat atteint ou résilié)
```

Vous pouvez réactiver un contrat suspendu. Une fois qu'un contrat est défini sur **Terminé**, il est en lecture seule.

---

## Cartes KPI

En haut de la page, deux cartes récapitulatives montrent la position actuelle :

- **Revenu mensuel contractuel total** — somme de tous les forfaits mensuels fixes actifs
- **Contrats actifs** — nombre de contrats actuellement au statut Actif

---

## Graphique de revenu

Un graphique à barres sous les cartes KPI ventile le revenu mensuel contractuel par client. Cela facilite l'identification des clients représentant votre base de revenu garantie et le repérage de ceux qui se terminent bientôt.

---

## Exporter

Utilisez **Exporter CSV** pour télécharger tous les contrats avec leurs champs, pour usage dans votre système comptable ou pour révision interne.

---

<Note>
Le revenu des contrats de capacité ne génère pas automatiquement de factures. Utilisez le module Facturation pour créer des factures référençant un contrat — référencez le contrat dans les notes de facture pour la traçabilité.
</Note>
