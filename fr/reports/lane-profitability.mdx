## Vue d'ensemble

Le rapport de rentabilité par ligne décompose vos performances financières par itinéraire. Il vous montre quelles lignes génèrent les meilleures marges et lesquelles coûtent plus qu'elles ne rapportent.

Accédez à **Rapports → Rentabilité par ligne**.

---

## Colonnes du rapport

| Colonne | Description |
|---|---|
| **Ligne** | Pays ou ville d'origine → Pays ou ville de destination |
| **Nb commandes** | Nombre de commandes sur cette ligne |
| **Revenu total** | Revenus cumulés de toutes les commandes |
| **Coût total** | Coûts variables (carburant, péage) + coûts fixes alloués |
| **Marge brute** | Revenu - Coûts variables |
| **Marge nette** | Revenu - Coûts totaux |
| **% Marge** | Marge nette en pourcentage du revenu |
| **CO2 total** | Émissions cumulées pour cette ligne |
| **Livraison à l'heure** | Pourcentage de livraisons dans les délais |

---

## Granularité

Basculez entre :
- **Par pays** — Vue agrégée (ex. : Roumanie → Allemagne)
- **Par ville** — Vue détaillée (ex. : Bucarest → Munich)

---

## Filtres

Utilisez les filtres pour analyser :
- Une période spécifique (mois, trimestre, année)
- Un type de remorque spécifique
- Un client spécifique

---

## Export

Cliquez sur **Exporter CSV** pour télécharger les données brutes pour analyse approfondie dans Excel. Le fichier inclut l'encodage UTF-8 BOM pour la compatibilité avec Excel.

<Note>
Le rapport est limité à 500 lignes par requête pour les performances. Si votre historique est plus important, exportez par périodes.
</Note>
