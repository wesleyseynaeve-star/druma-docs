## Vue d'ensemble

Chaque document rattaché à une commande est disponible pour votre client directement depuis son portail. Cela supprime les allers-retours d'envoi de PDF par email sur demande — le client obtient ce dont il a besoin, quand il en a besoin, sans vous contacter.

## Types de documents disponibles

Selon ce qui a été téléversé ou rattaché à une commande, les clients peuvent voir les documents suivants :

| Document | Ce que c'est | Qui le téléverse |
|---|---|---|
| **PDF eCMR signé** | Lettre de voiture électronique certifiée et légalement valide | Généré automatiquement quand le chauffeur termine le flux eCMR |
| **Scan CMR** | Photo du document CMR papier | Le chauffeur photographie le CMR papier dans l'application chauffeur |
| **Photos POD** | Photographies de preuve de livraison | Le chauffeur les téléverse au point de livraison |
| **Bons de livraison** | Toute note spécifique à la livraison ou document de remise | Le chauffeur ou le planificateur les rattache pendant ou après la livraison |
| **Tickets de pesée** | Certificat de poids d'un pont-bascule | Le chauffeur le téléverse au pont-bascule |
| **Factures** | La facture commerciale du transport | Le planificateur la génère et l'envoie via Druma |

<Note>
Les documents n'apparaissent dans le portail qu'une fois téléversés par le chauffeur ou rattachés par le planificateur. Si un client signale qu'un document manque, vérifiez d'abord si le chauffeur a bien terminé cette étape dans l'application chauffeur, ou si vous devez le rattacher manuellement depuis la page de détail de la commande dans Druma.
</Note>

## Comment télécharger un document

<Steps>
  <Step title="Ouvrez la commande">
    Dans le portail, trouvez la commande concernée dans la liste. Cliquez sur la référence de commande pour ouvrir la vue de détail.
  </Step>
  <Step title="Accédez à l'onglet Documents">
    Dans la commande, cliquez sur l'onglet **Documents**. Tous les documents disponibles sont listés avec leur type, leur date de téléversement et leur nom de fichier.
  </Step>
  <Step title="Cliquez sur le nom du document">
    Cliquez sur le nom du document pour ouvrir un aperçu PDF ou accéder directement au fichier, selon le type de document.
  </Step>
  <Step title="Cliquez sur Télécharger">
    Cliquez sur le bouton **Télécharger** pour enregistrer le fichier sur votre appareil.
  </Step>
</Steps>

## Remarques sur des types de documents spécifiques

### eCMR signé

Le PDF eCMR signé est légalement valide et certifié selon le protocole eCMR. Il est généré automatiquement une fois que le chauffeur termine le processus de signature électronique — aucune action manuelle n'est requise de votre part. Si une expédition a utilisé un CMR papier à la place, cherchez le document **Scan CMR**.

### Factures

Les factures n'apparaissent dans le portail qu'une fois que le planificateur a généré la facture dans Druma **et** cliqué sur **Envoyer** pour la remettre au client. Une facture encore au statut Brouillon n'est absolument pas visible pour le client.

<Warning>
Si un client ne trouve pas sa facture, vérifiez dans **Facturation** si le statut de la facture est Brouillon ou Envoyée. Les factures en brouillon ne sont pas visibles dans le portail. Ouvrez la facture et cliquez sur **Envoyer** pour la faire apparaître pour le client.
</Warning>

### Photos POD

Les photos de preuve de livraison sont téléversées par le chauffeur au point de livraison. Si une livraison est marquée comme terminée mais que les photos manquent, contactez le chauffeur — il devra peut-être les téléverser rétroactivement depuis l'application chauffeur.
