## Vue d'ensemble

SmartBill est une plateforme de facturation cloud populaire en Roumanie. Avec cette intégration, chaque facture confirmée dans Druma est automatiquement poussée vers SmartBill — aucune ressaisie manuelle.

---

## Configuration

Voir [Configuration des intégrations → SmartBill](/fr/admin/integration-setup) pour les instructions de configuration.

---

## Comment ça fonctionne

Lorsque vous générez et confirmez une facture dans Druma :

1. Druma appelle automatiquement l'API SmartBill.
2. SmartBill crée la facture correspondante dans votre compte.
3. La facture apparaît dans votre liste SmartBill avec le même numéro et les mêmes montants.
4. Druma enregistre le statut de synchronisation SmartBill dans le panneau de détails de la facture.

---

## Pousser manuellement une facture

Si une facture n'a pas été synchronisée automatiquement (ex. : erreur de connexion temporaire), vous pouvez la pousser manuellement :

<Steps>
  <Step title="Ouvrez la facture">
    Accédez à la facture dans Druma.
  </Step>
  <Step title="Cliquez sur Envoyer vers SmartBill">
    Cliquez sur le bouton **Envoyer vers SmartBill** dans le panneau de détails.
  </Step>
</Steps>

---

## Résolution des problèmes

Si le statut SmartBill affiche une erreur :

1. Vérifiez que votre abonnement SmartBill est actif.
2. Vérifiez que votre clé API n'a pas été régénérée dans SmartBill.
3. Si la clé a changé, mettez-la à jour dans **Paramètres → Intégrations → SmartBill**.
