## Vue d'ensemble

TransFollow est le fournisseur d'eCMR certifié qui alimente les fonctionnalités d'eCMR de Druma. TransFollow est accrédité en tant que fournisseur eCMR et ses documents sont juridiquement valables dans tous les pays ayant ratifié le Protocole e-CMR.

Voir [Configuration des intégrations → TransFollow](/fr/admin/integration-setup) pour les instructions de configuration étape par étape.

## Mode simulation

Le mode simulation vous permet de tester le flux eCMR complet sans émettre de vrais documents légaux. Utilisez-le pour :
- Former de nouveaux chauffeurs au processus de signature
- Tester la configuration avant de passer en production
- Démonstrations clients

<Warning>
Désactivez le mode simulation avant de traiter de vraies expéditions.
</Warning>

---

## Flux de bout en bout

1. **Création** — Druma crée l'eCMR via l'API TransFollow lors de la confirmation de commande.
2. **Signature expéditeur** — Le chauffeur fait signer l'expéditeur au chargement.
3. **Signature chauffeur** — Le chauffeur signe immédiatement après.
4. **Signature destinataire** — Le destinataire signe à la livraison.
5. **Certification** — TransFollow émet le PDF certifié.
6. **Stockage** — Druma stocke le PDF dans la commande automatiquement.

---

## PDF certifié

Le PDF certifié inclut :
- Données complètes CMR (expéditeur, transporteur, destinataire, marchandises)
- Horodatages de toutes les signatures
- Hachages cryptographiques prouvant l'authenticité
- QR code pour vérification en ligne
