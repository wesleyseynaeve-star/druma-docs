## Vue d'ensemble

SAGA et WinMENTOR sont les deux logiciels de comptabilité les plus répandus en Roumanie. Druma exporte les données comptables dans des formats directement compatibles avec ces logiciels.

---

## SAGA Classic et WinMENTOR Classic

Aucune clé API n'est nécessaire. Druma génère des fichiers d'export structurés que vous importez directement dans votre logiciel comptable.

Accédez à **Rapports → Export comptable**, sélectionnez la plage de dates et le format comptable, et téléchargez le fichier.

---

## Contenu de l'export

Le fichier ZIP d'export contient :
- **INTRARI.dbf** — Factures d'achat
- **IESIRI.dbf** — Factures de vente
- **INCASARI.xml** — Données de paiement (format XML Payments)

---

## WinMENTOR Enterprise (API REST)

WinMENTOR Enterprise prend en charge l'intégration directe via API.

<Steps>
  <Step title="Ouvrez la carte WinMENTOR Enterprise">
    Dans **Paramètres → Intégrations**, cliquez sur **WinMENTOR Enterprise**.
  </Step>
  <Step title="Saisissez l'endpoint REST">
    Saisissez l'URL de l'endpoint API REST fourni par votre administrateur WinMENTOR Enterprise.
  </Step>
  <Step title="Saisissez les identifiants">
    Saisissez le nom d'utilisateur et le mot de passe API configurés dans WinMENTOR Enterprise.
  </Step>
  <Step title="Testez et enregistrez">
    Cliquez sur **Tester la connexion**, puis **Enregistrer**.
  </Step>
</Steps>
