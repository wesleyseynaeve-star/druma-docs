Druma est responsable du traitement des données que votre entreprise stocke sur la plateforme.

---

## Données stockées par Druma

### Données d'entreprise et de compte
- Nom de l'entreprise, CIF, adresse, coordonnées
- Comptes utilisateurs : nom, adresse email, rôle, horodatages de connexion

### Données opérationnelles
- Commandes : itinéraire, marchandises, adresses, historique des statuts, chauffeur et véhicule assignés
- Fiches clients : nom, adresse, personne de contact, CIF
- Informations chauffeur : nom, contact, numéro de permis, détails ADR
- Fiches véhicules : plaque, marque/modèle, norme Euro

### Données financières
- Enregistrements de factures, montants, statut de paiement
- Données d'import de cartes carburant

### Documents eCMR
- Lettres de voiture électroniques émises via TransFollow

### Données de position GPS
- Druma stocke **uniquement la position GPS actuelle**. **Aucun historique de position n'est stocké.** La précédente est écrasée dès qu'une nouvelle est reçue.

### Journaux d'audit
- Un journal des actions utilisateur importantes

<Note>
Druma ne stocke pas les données de tachographe, les enregistrements d'appels, les données biométriques ni les informations médicales des chauffeurs.
</Note>

## Périodes de conservation des données

| Catégorie de données | Période de conservation |
|---|---|
| Données opérationnelles | Durée de l'abonnement + 90 jours |
| Données de compte utilisateur | Durée de l'abonnement + 90 jours |
| Journaux d'audit | 2 ans |
| Documents eCMR | 10 ans (exigence légale du Protocole e-CMR) |
| Données de position GPS | Écrasées en temps réel — aucun historique |
| Dossiers de facturation | 7 ans (loi comptable) |

## Demande d'export de données

<Steps>
  <Step title="Accédez à Paramètres → Données et confidentialité">
    Seuls les rôles Admin et Admin entreprise peuvent accéder à cette section.
  </Step>
  <Step title="Cliquez sur Demander un export">
    Druma préparera une archive ZIP de toutes les données de votre entreprise en JSON et CSV.
  </Step>
  <Step title="Recevez l'export">
    Vous recevrez un email avec un lien de téléchargement dans les **24 heures**. Le lien expire après 7 jours.
  </Step>
</Steps>

---

## Suppression de votre compte

<Warning>
La suppression du compte est permanente et irréversible après la fenêtre de conservation de 90 jours. Téléchargez toutes les données dont vous avez besoin avant de demander la suppression.
</Warning>

## Vos droits RGPD

- **Droit d'accès** — demander une copie des données personnelles vous concernant
- **Droit de rectification** — corriger les données inexactes
- **Droit à l'effacement** — demander la suppression (sous réserve des obligations légales)
- **Droit à la portabilité** — recevoir vos données dans un format structuré
- **Droit à la limitation** — demander à Druma de limiter le traitement
- **Droit d'opposition** — s'opposer au traitement basé sur des intérêts légitimes

Pour exercer l'un de ces droits, envoyez un email à **privacy@druma.io**. Druma répondra dans les 30 jours.
