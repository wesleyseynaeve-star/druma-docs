Les paramètres **Marque et formats** contrôlent la façon dont votre entreprise apparaît sur chaque document généré par Druma.

**Qui peut modifier :** Uniquement les rôles Admin et Admin entreprise.

---

## Accéder aux paramètres

Accédez à **Paramètres → Marque et formats**. La page comporte trois onglets : **Marque**, **Format des factures** et **Format des commandes**.

---

## Onglet Marque

### Logo

Cliquez sur **Télécharger le logo** pour sélectionner un fichier image (PNG, JPG ou SVG). Le logo apparaît dans le coin supérieur gauche des factures PDF.

Pour de meilleurs résultats, utilisez un logo avec fond transparent d'une largeur minimale de 300 px.

### Couleur principale

Cliquez sur le sélecteur de couleur **Couleur principale** et saisissez votre couleur de marque.

### Couleur d'accentuation

Utilisée pour les éléments secondaires tels que les en-têtes de tableaux sur les PDF.

### Enregistrement

Cliquez sur **Enregistrer la marque**.

<Note>
Les modifications de marque s'appliquent aux **nouveaux** documents uniquement. Les factures déjà générées ne sont pas mises à jour rétroactivement.
</Note>

## Onglet Format des factures

### Jetons de format

| Jeton | Ce qu'il insère |
|---|---|
| `{PREFIX}` | Un préfixe de texte fixe que vous définissez |
| `{YYYY}` | Année sur quatre chiffres (ex. : `2026`) |
| `{YY}` | Année sur deux chiffres (ex. : `26`) |
| `{MM}` | Mois sur deux chiffres (ex. : `04`) |
| `{NNN}` | Numéro séquentiel sur trois chiffres, complété par des zéros |
| `{NNNN}` | Numéro séquentiel sur quatre chiffres |

**Format par défaut :** `DR-{YYYY}-{NNN}` → produit `DR-2026-001`, `DR-2026-002`...

### Format du numéro d'avoir

Les avoirs utilisent un compteur séparé. **Format par défaut :** `CN-{YYYY}-{NNN}`

### Enregistrement

Cliquez sur **Enregistrer le format des factures**.

<Warning>
Changer le format en milieu d'année ne renumérotera pas les factures existantes.
</Warning>

---

## Onglet Format des commandes

Définit le modèle de numérotation pour les numéros de référence des commandes.

**Format par défaut :** `DR-{YYYY}-{NNN}`

Cliquez sur **Enregistrer le format des commandes**.
