## Vue d'ensemble

Druma peut recevoir les factures de vos fournisseurs (sous-traitants, prestataires de carburant, ateliers) par email et les acheminer vers une file de révision. L'IA extrait les informations clés et tente de les faire correspondre automatiquement avec les commandes d'achat existantes.

Accédez à **Finance → Transfert → File de factures** pour voir la file d'attente.

## Examiner une facture reçue

<Steps>
  <Step title="Ouvrez la file d'attente">
    Accédez à **Finance → Transfert → File de factures**.
  </Step>
  <Step title="Cliquez sur une facture en attente">
    Ouvrez la facture à examiner. Vous verrez les données extraites par l'IA : fournisseur, montant, date, numéro de facture.
  </Step>
  <Step title="Vérifiez la correspondance automatique">
    Si l'IA a trouvé une commande correspondante, vérifiez que c'est correct.
  </Step>
  <Step title="Approuvez ou réaffectez">
    - **Approuver** — La facture est marquée comme reçue et liée à la commande.
    - **Réaffecter** — Trouvez manuellement la bonne commande.
    - **Rejeter** — La facture n'est pas valide ou en double.
  </Step>
</Steps>
