## Vue d'ensemble

Druma exporte les données comptables dans des formats compatibles avec SAGA et WinMENTOR. Cela vous permet de maintenir votre comptabilité officielle dans votre logiciel habituel tout en gérant les opérations dans Druma.

---

## Créer un export SAGA

<Steps>
  <Step title="Accédez à Rapports → Export comptable">
    Cliquez sur **Nouvel export**.
  </Step>
  <Step title="Sélectionnez la plage de dates">
    Choisissez la période pour laquelle vous souhaitez exporter les données (ex. : le mois dernier).
  </Step>
  <Step title="Sélectionnez le format">
    Choisissez **SAGA Classic**, **SAGA Enterprise** ou **WinMENTOR** selon votre logiciel.
  </Step>
  <Step title="Téléchargez l'export">
    Cliquez sur **Générer l'export**. Un fichier ZIP se télécharge contenant :
    - **INTRARI.dbf** — Factures d'achat (fournisseurs)
    - **IESIRI.dbf** — Factures de vente (clients)
    - **INCASARI.xml** — Données de paiement
  </Step>
</Steps>

---

## Importer dans SAGA

1. Ouvrez SAGA Classic ou Enterprise.
2. Accédez à **Import → Import Druma**.
3. Sélectionnez les fichiers du ZIP téléchargé.
4. SAGA valide et importe les données.
