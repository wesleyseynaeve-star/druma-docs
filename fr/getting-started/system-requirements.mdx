## Vue d'ensemble

Druma est une plateforme web — il n'y a aucun logiciel à installer. Tout ce dont vous avez besoin est un navigateur moderne et une connexion Internet fiable.

## Navigateurs pris en charge (plateforme web)

| Navigateur | Version minimale |
|---------|----------------|
| Google Chrome | 90 ou supérieur |
| Mozilla Firefox | 90 ou supérieur |
| Microsoft Edge | 90 ou supérieur |
| Apple Safari | 14 ou supérieur |

<Note>
Google Chrome est recommandé pour la meilleure expérience, notamment pour le tableau de planification et les fonctionnalités d'aperçu PDF.
</Note>

<Warning>
Internet Explorer n'est pas pris en charge. Installez Chrome ou Edge avant d'utiliser Druma.
</Warning>

### Paramètres de navigateur requis

- **JavaScript doit être activé.** Activé par défaut dans tous les navigateurs modernes.
- **Les cookies doivent être autorisés** pour `druma.io`. Druma utilise des cookies pour maintenir votre session.
- **Les fenêtres contextuelles de druma.io doivent être autorisées.** Druma ouvre les rapports PDF dans un nouvel onglet.

## Recommandations de taille d'écran

| Rôle | Écran recommandé |
|------|-------------------|
| Planificateur (tableau de planification) | Bureau ou ordinateur portable, 1920px de large ou plus |
| Planificateur (utilisation générale) | Bureau ou ordinateur portable, minimum 1280px de large |
| Admin entreprise / Admin | Bureau ou ordinateur portable, minimum 1280px de large |
| Service client | Bureau ou ordinateur portable, minimum 1280px de large |
| Chauffeur | Tout smartphone moderne |

Le tableau de planification est plus facile à utiliser sur un grand écran. Sur téléphone ou tablette, il est disponible en mode lecture seule.

## Application chauffeur (PWA)

Les chauffeurs n'utilisent pas la plateforme web. Chaque chauffeur dispose d'un lien personnel qui ouvre l'**application chauffeur Druma** dans le navigateur de son téléphone.

### Téléphones Android

1. Ouvrez le lien dans **Google Chrome**.
2. Appuyez sur la bannière « Ajouter à l'écran d'accueil ».
3. L'icône de l'application apparaît sur l'écran d'accueil.

### iPhone et iPad (iOS)

1. Ouvrez le lien dans **Safari** (pas Chrome).
2. Appuyez sur le bouton **Partager** (le carré avec une flèche).
3. Appuyez sur **Sur l'écran d'accueil**, puis **Ajouter**.

<Note>
Sur iPhone, l'application doit être ouverte dans Safari pour l'installation. Si le chauffeur utilise Chrome, il doit copier-coller le lien dans Safari.
</Note>

## Connexion Internet

Druma nécessite une connexion Internet active en permanence. **Il n'y a pas de mode hors ligne.**
