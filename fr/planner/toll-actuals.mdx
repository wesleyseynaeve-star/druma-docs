## Vue d'ensemble

Druma calcule les coûts de péage estimés pour chaque commande via l'API de routage HERE. Ces estimations sont basées sur le poids du véhicule, le nombre d'essieux et l'itinéraire.

---

## Coûts de péage dans les commandes

Lors de la création d'une commande, Druma calcule automatiquement :

- **Distance totale** (km)
- **Coût de péage estimé** (€)
- **Émissions CO2** (kg)

Ces chiffres apparaissent dans la section **Itinéraire** du formulaire de commande et dans le panneau de détails.

---

## Impact sur la marge

Les coûts de péage sont inclus dans le calcul de marge de la commande :

**Marge = Prix convenu - Coûts variables (carburant, péage) - Coûts fixes alloués**

---

## Données de péage dans les rapports

Les coûts de péage agrégés apparaissent dans :

- **Rapports → Rentabilité par ligne** — Coûts de péage par itinéraire.
- **Rapports → CO2** — En combinaison avec les données d'émissions.
