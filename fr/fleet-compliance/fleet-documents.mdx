## Vue d'ensemble

Druma suit les dates d'expiration de tous les documents réglementaires de votre flotte. Vous recevez des alertes avant qu'un document n'expire afin de ne jamais être pris par surprise lors d'un contrôle routier ou d'une inspection.

Accédez à **Flotte → Documents** pour voir le tableau de bord des documents.

---

## Types de documents suivis

| Document | Description |
|---|---|
| **ITP / Contrôle technique** | Inspection technique périodique du véhicule |
| **Assurance RCA** | Assurance responsabilité civile automobile |
| **CEMT / Licence de transport** | Licence de transport routier international |
| **Certificat ADR** | Autorisation pour le transport de marchandises dangereuses |
| **Carte verte** | Assurance internationale |
| **Tachygraphe** | Calibrage du tachygraphe numérique |
| **Permis de conduire** | Permis du chauffeur |
| **CPC** | Certificat de compétence professionnelle du chauffeur |

## Ajouter un document

<Steps>
  <Step title="Ouvrez le profil du véhicule ou du chauffeur">
    Accédez à Flotte → Véhicules ou Flotte → Chauffeurs et cliquez sur l'entrée.
  </Step>
  <Step title="Accédez à l'onglet Documents">
    Cliquez sur l'onglet **Documents**.
  </Step>
  <Step title="Cliquez sur Ajouter un document">
    Sélectionnez le type de document et saisissez la date d'expiration.
  </Step>
  <Step title="Téléchargez une copie numérique (optionnel)">
    Téléchargez une copie PDF ou image du document pour référence.
  </Step>
  <Step title="Enregistrer">
    Cliquez sur **Enregistrer**.
  </Step>
</Steps>

---

## Alertes d'expiration

Druma envoie des alertes lorsqu'un document expire dans :
- **30 jours** — avertissement précoce
- **7 jours** — avertissement urgent
- **0 jours (expiré)** — alerte critique

Les documents expirés apparaissent avec un badge rouge sur le tableau de planification lors de l'affectation d'un camion.
