## Vue d'ensemble

ADR est l'accord européen sur le transport international de marchandises dangereuses par route. Les chauffeurs et les véhicules transportant des marchandises dangereuses doivent tous deux détenir des certificats ADR valides.

Druma vérifie automatiquement les certificats ADR avant d'autoriser l'affectation d'un camion à une commande ADR.

---

## Enregistrer un certificat ADR

**Pour un chauffeur :**
Accédez à Flotte → Chauffeurs → profil du chauffeur → onglet Documents → Ajouter → **Certificat ADR chauffeur**.

**Pour un véhicule :**
Accédez à Flotte → Véhicules → profil du véhicule → onglet Documents → Ajouter → **Certificat ADR véhicule**.

Saisissez la date d'expiration et téléchargez une copie du certificat.

---

## Vérifications ADR sur le tableau de planification

Lorsque vous affectez un camion à une commande marquée ADR, Druma vérifie :

1. Le véhicule a un certificat ADR valide
2. Le chauffeur principal a un certificat ADR valide
3. Si un deuxième chauffeur est assigné, il a également un certificat ADR valide

**Résultat de la vérification :**

| Résultat | Affichage |
|---|---|
| ADR valide pour les deux | Affectation autorisée |
| ADR expirant dans 7 jours | Avertissement (affectation autorisée) |
| ADR expiré ou manquant | Erreur (affectation bloquée) |

## Classes ADR

Druma prend en charge toutes les classes de marchandises dangereuses ADR :

| Classe | Description |
|---|---|
| Classe 1 | Explosifs |
| Classe 2 | Gaz |
| Classe 3 | Liquides inflammables |
| Classe 4 | Solides inflammables |
| Classe 5 | Substances comburantes et peroxydes |
| Classe 6 | Substances toxiques et infectieuses |
| Classe 7 | Matières radioactives |
| Classe 8 | Substances corrosives |
| Classe 9 | Autres substances dangereuses |
