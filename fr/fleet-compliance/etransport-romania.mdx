## Vue d'ensemble

Le système e-Transport est une exigence réglementaire roumaine pour le transport de certains types de marchandises à haute valeur sur le territoire roumain. Les transporteurs doivent déclarer les chargements qualifiés à l'ANAF avant le départ et transmettre des données GPS pendant le transit.

---

## Quand e-Transport s'applique-t-il ?

e-Transport s'applique aux commandes où :
- La marchandise a une valeur élevée (au-dessus du seuil défini par l'ANAF)
- L'itinéraire inclut le territoire roumain
- La marchandise appartient à certaines catégories (produits alimentaires, produits de construction, minéraux, etc.)

Druma détecte automatiquement si une commande nécessite une déclaration e-Transport en fonction de la valeur des marchandises et de l'itinéraire.

---

## Configuration

<Steps>
  <Step title="Configurez vos identifiants ANAF">
    Accédez à **Paramètres → Intégrations → ANAF e-Transport** et saisissez votre certificat numérique et vos identifiants ANAF.
  </Step>
  <Step title="Activez la déclaration automatique">
    Activez **Déclaration automatique e-Transport** dans les paramètres de l'intégration.
  </Step>
</Steps>

---

## Processus de déclaration

1. Druma détecte qu'une commande nécessite une déclaration e-Transport.
2. La déclaration XML est soumise automatiquement à l'ANAF avant le départ.
3. L'ANAF renvoie un **code UIT** (Unique Identification Token) — c'est le numéro de déclaration.
4. Druma affiche le code UIT dans l'application chauffeur.
5. Pendant le transit, Druma transmet automatiquement les données GPS du chauffeur à l'ANAF.

---

## Code UIT pour les chauffeurs

Le chauffeur voit le code UIT dans son application sous forme de carte ambrée avec un grand code en police monospace. Il peut copier le code et le présenter aux autorités si demandé.

<Warning>
Le chauffeur ne peut pas partir sans UIT valide si la commande est soumise à e-Transport. Si le code UIT n'est pas reçu avant le départ prévu, contactez votre répartiteur.
</Warning>
