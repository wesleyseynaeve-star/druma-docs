## Vue d'ensemble

L'onglet Pneus dans Fleet suit le montage des pneus, l'état et l'historique de remplacement pour chaque véhicule.

**Accès :** Flotte → Camions → ouvrez un véhicule → onglet **Pneus**.

---

## Enregistrements de pneus

| Champ | Description |
|---|---|
| **Position** | Essieu et côté (ex. Avant Gauche, Essieu Moteur Droit Intérieur) |
| **Marque et modèle** | Fabricant et produit |
| **Dimension** | Désignation (ex. 315/70 R22.5) |
| **Numéro de série** | Code DOT du pneu |
| **Date de montage** | Quand le pneu a été installé |
| **Profondeur de bande (mm)** | Profondeur mesurée actuelle |
| **État** | Bon / Surveiller / Remplacer |
| **Notes** | Notes en texte libre |

---

## Alertes d'état des pneus

| État | Alerte |
|---|---|
| **Surveiller** | Indicateur jaune sur la vignette du véhicule |
| **Remplacer** | Indicateur rouge ; alerte email au Gestionnaire Flotte et Admin |

<Warning>
Une profondeur de bande inférieure à **3 mm** déclenche automatiquement un indicateur **Remplacer** — en dessous du minimum légal UE pour les véhicules commerciaux.
</Warning>

---

## Suivi des coûts des pneus

Lors du remplacement d'un pneu : éditez l'enregistrement → définissez l'état sur **Remplacé** → entrez le **Coût de remplacement (€)** → Enregistrez. Le coût s'additionne dans le rapport de coût du véhicule.
