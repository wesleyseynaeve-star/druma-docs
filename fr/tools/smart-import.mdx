## Vue d'ensemble

L'Import intelligent vous permet d'alimenter Druma en données en masse en téléchargeant vos documents existants. L'IA (Google Gemini) analyse les documents et extrait automatiquement les informations structurées avec un score de confiance par champ.

Accédez à **Outils → Import intelligent** pour commencer.

---

## Formats pris en charge

| Format | Description |
|---|---|
| **PDF** | Certificats d'immatriculation, certificats d'assurance, permis de conduire |
| **Images** | JPEG, PNG, WebP, HEIC — photos de documents |
| **CSV / XLSX** | Feuilles de calcul de flotte, listes de clients |

---

## Processus d'import

<Steps>
  <Step title="Téléchargez vos documents">
    Faites glisser et déposez vos documents dans la zone de téléchargement, ou cliquez pour sélectionner des fichiers.
  </Step>
  <Step title="Attendez l'extraction">
    Gemini analyse chaque document et extrait les données. Cela prend généralement quelques secondes par document.
  </Step>
  <Step title="Examinez les données extraites">
    Le tableau de bord affiche les données extraites par catégorie (Flotte, Chauffeurs, Clients, Assurances) avec un score de confiance pour chaque champ.
    - **Vert (haute confiance)** — données probablement correctes
    - **Ambre (confiance moyenne)** — vérifiez avant d'importer
    - **Rouge (faible confiance)** — corrigez manuellement
  </Step>
  <Step title="Modifiez si nécessaire">
    Cliquez sur n'importe quel champ pour le modifier directement dans le tableau.
  </Step>
  <Step title="Vérifiez les doublons">
    Druma vérifie automatiquement les enregistrements existants (plaques, numéros de TVA) et signale les doublons potentiels.
  </Step>
  <Step title="Importez">
    Sélectionnez les catégories à importer et cliquez sur **Importer les données sélectionnées**.
  </Step>
</Steps>
