## Vue d'ensemble

Importez tous vos sous-traitants en une seule fois depuis une feuille de calcul. Après l'import, chaque sous-traitant reçoit automatiquement un token de portail.

Accédez à **Sous-traitants → Importer** pour commencer.

---

## Colonnes du modèle

| Colonne | Obligatoire | Description |
|---|---|---|
| `company_name` | Oui | Nom de l'entreprise sous-traitante |
| `vat_number` | Oui | Numéro de TVA (utilisé pour la déduplication) |
| `country` | Oui | Code pays ISO 2 lettres |
| `contact_name` | Non | Nom du contact principal |
| `contact_email` | Non | Email du contact (pour l'envoi du lien portail) |
| `contact_phone` | Non | Téléphone du contact |
| `address` | Non | Adresse complète |

---

## Token de portail

Chaque sous-traitant importé reçoit automatiquement un token de portail valable 30 jours. Vous pouvez :
- Envoyer le lien de portail par email au contact importé
- Copier le lien depuis le profil du sous-traitant
