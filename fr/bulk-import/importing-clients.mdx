## Vue d'ensemble

Importez tous vos clients en une seule fois depuis une feuille de calcul existante. Druma détecte automatiquement les clients en double en comparant les numéros de TVA.

Accédez à **Clients → Importer** pour commencer.

---

## Colonnes du modèle

| Colonne | Obligatoire | Description |
|---|---|---|
| `company_name` | Oui | Nom de l'entreprise |
| `vat_number` | Oui | Numéro de TVA (utilisé pour la déduplication) |
| `country` | Oui | Code pays ISO 2 lettres (ex. : FR, BE, RO) |
| `address` | Non | Adresse complète |
| `city` | Non | Ville |
| `postal_code` | Non | Code postal |
| `contact_name` | Non | Nom du contact principal |
| `contact_email` | Non | Email du contact |
| `contact_phone` | Non | Téléphone du contact |
| `payment_terms_days` | Non | Conditions de paiement en jours |
| `currency` | Non | Devise par défaut (EUR, RON, etc.) |

---

## Déduplication

Druma compare le numéro de TVA de chaque ligne avec les clients existants :
- Si le TVA existe déjà, la ligne est ignorée (pas de mise à jour)
- Si le TVA est nouveau, le client est créé

## Processus d'import

<Steps>
  <Step title="Téléchargez et remplissez le modèle">
    Cliquez sur **Télécharger le modèle** et remplissez-le avec vos données clients.
  </Step>
  <Step title="Téléchargez le fichier">
    Cliquez sur **Importer CSV/XLSX** et sélectionnez votre fichier.
  </Step>
  <Step title="Examinez et confirmez">
    Vérifiez l'aperçu et confirmez l'import.
  </Step>
</Steps>
