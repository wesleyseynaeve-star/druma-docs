## Übersicht

Der nächtliche Datenqualitätsscan läuft automatisch um 09:00 Uhr UTC. Er prüft Ihr System auf häufige Probleme und zeigt Ergebnisse im Dashboard an, damit Ihr Team Probleme proaktiv beheben kann.

---

## Was geprüft wird

| Prüfung | Schweregrad |
|---|---|
| Aufträge ohne Preis | Kritisch |
| Rechnungen überfällig über 60 Tage | Kritisch |
| Fahrzeugdokumente abgelaufen | Kritisch |
| Fahrerdokumente abgelaufen | Kritisch |
| Aufträge ohne Fahrerzuweisung | Warnung |
| Fahrzeuge ohne Versicherung | Warnung |
| Fehlende Kundenkontakte | Info |
| Keine Aktivität für 7+ Tage | Info |
| Lieferungen ohne Liefernachweis | Warnung |

---

## Ergebnisse anzeigen

Gehen Sie zu **Tools → Datenqualität**, um:
- Alle gefundenen Probleme nach Schweregrad gruppiert zu sehen
- Auf einen Klick direkt zur betroffenen Ressource zu navigieren
- Probleme als gelöst oder ignoriert zu markieren

---

Admins und Unternehmensadministratoren erhalten eine tägliche E-Mail-Zusammenfassung kritischer Probleme.
