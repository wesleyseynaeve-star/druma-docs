## Was ist eine Preistabelle?

Eine Preistabelle ist eine gespeicherte Preisvorlage für eine Route, die Sie regelmäßig fahren. Wenn Sie jede Woche Ladung von Rumänien nach Deutschland transportieren, speichern Sie eine Preistabelle für diese Strecke — und Druma schlägt den Preis automatisch vor, wenn Sie einen neuen Auftrag auf dieser Route erstellen.

Preistabellen sind ein Ausgangspunkt, keine Regel. Planer können den vorgeschlagenen Preis bei einzelnen Aufträgen immer überschreiben.

---

## Eine Preistabelle erstellen

Gehen Sie zu **Einstellungen → Preistabellen** und klicken Sie auf **Preistabelle hinzufügen**.

### Name

Geben Sie einen klaren, beschreibenden Namen.

*Beispiele: "RO→DE Tautliner", "Rumänien nach Niederlande Frigo"*

### Ursprungsland und Zielland

Preistabellen stimmen auf Länderebene überein — eine Preistabelle deckt alle Städte innerhalb dieser Länder ab.

### Anhängertyp

Curtainsider, Kühlfahrzeug, Tieflader, Tankfahrzeug, Kofferaufbau, Mega-Trailer, Tieflader, Sonstige.

Druma schlägt eine Preistabelle nur vor, wenn der Anhängertyp übereinstimmt.

### Preistyp

- **Pauschalpreis** — Fester Betrag unabhängig von der Entfernung.
- **Pro Kilometer** — Tarif multipliziert mit der HERE-Routendistanz.

### Preis und Währung

Betrag und Währung (EUR, RON, USD, GBP usw.).

### Kraftstoffzuschlag %

Wenn Sie einen Kraftstoffzuschlag aufschlagen, geben Sie den Prozentsatz ein. Druma addiert ihn zum Grundpreis.

*Beispiel: €1.200 Grundpreis + 8% = €1.296 vorgeschlagener Preis*

### Notizen

Interne Notizen — nur für Planer sichtbar, nicht auf Dokumenten.

---

## Eine Preistabelle bearbeiten

**Einstellungen → Preistabellen** → Bearbeitungssymbol (Stift) → Änderungen → **Speichern**.

Der aktualisierte Preis gilt ab sofort für neue Aufträge.

---

## Eine Preistabelle deaktivieren

Schalten Sie den **Aktiv**-Schalter aus. Deaktivierte Preistabellen werden nicht mehr vorgeschlagen, bleiben aber zur Referenz in der Liste.

<Note>
Deaktivieren ist besser als Löschen, besonders wenn Sie diese Strecke später wieder aktivieren möchten.
</Note>
