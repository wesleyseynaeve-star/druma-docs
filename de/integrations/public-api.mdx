## Übersicht

Die Druma Public API ermöglicht es Entwicklern, Druma-Daten in externe Systeme zu integrieren. Sie können Auftragserstelle-, Statusabfrage- und Eventbenachrichtigungs-Workflows aufbauen.

---

## Authentifizierung

Die API verwendet Bearer-Token-Authentifizierung. Generieren Sie Ihren API-Schlüssel in **Einstellungen → Integrationen → API**.

```http
Authorization: Bearer your_api_key
```

---

## Basis-URL

```
https://api.druma.io/v1/
```

---

## Endpunkte

| Endpunkt | Methode | Beschreibung |
|---|---|---|
| `/orders` | GET | Aufträge mit Filtern auflisten |
| `/orders/{id}` | GET | Auftragsdetails abrufen |
| `/orders` | POST | Neuen Auftrag erstellen |
| `/orders/{id}/status` | PATCH | Auftragsstatus aktualisieren |
| `/vehicles` | GET | Fahrzeugliste abrufen |
| `/drivers` | GET | Fahrerliste abrufen |

---

## Webhooks

Konfigurieren Sie Webhook-Endpunkte, um Echtzeitbenachrichtigungen zu erhalten, wenn:
- Ein Auftrag den Status ändert
- Ein Auftrag zugestellt wird
- Eine Rechnung gesendet wird

Konfigurieren Sie Webhooks in **Einstellungen → Integrationen → Webhooks**.
