## Wann eine Verspätung melden?

Melden Sie eine Verspätung, wenn Sie wissen, dass Sie den Abholtermin oder die Lieferzeit deutlich verpassen werden — typischerweise mehr als 30–60 Minuten. Häufige Ursachen:
- Stau oder Unfälle
- Schlechte Wetterverhältnisse
- Verzögerungen an vorherigen Stopps
- Fahrzeugpannen

---

## So melden Sie eine Verspätung

<Steps>
  <Step title="Auftragsbildschirm öffnen">
    Tippen Sie auf die Schaltfläche **Verspätung melden** auf dem Auftragsbildschirm.
  </Step>
  <Step title="Grund auswählen">
    Wählen Sie den Verspätungsgrund:
    - Stau / Verkehr
    - Schlechtes Wetter
    - Fahrzeugproblem
    - Verzögerung beim vorherigen Stopp
    - Sonstiges
  </Step>
  <Step title="Neue geschätzte Ankunftszeit eingeben">
    Geben Sie ein, wann Sie nun ankommen werden.
  </Step>
  <Step title="Senden">
    Tippen Sie auf **Verspätung melden**. Ihr Disponent wird sofort benachrichtigt.
  </Step>
</Steps>

---

## Was passiert nach dem Melden?

- Ihr Disponent erhält eine Warnmeldung im Dashboard
- Der Auftrag wird auf der Planungstafel mit einem Verspätungs-Badge markiert
- Optionally: Der Kunde erhält eine automatische E-Mail mit der aktualisierten ETA
