## Übersicht

Druma ist eine webbasierte Plattform — es muss keine Software auf Ihrem Computer installiert werden. Sie benötigen lediglich einen modernen Browser und eine zuverlässige Internetverbindung. Diese Seite listet auf, was unterstützt wird und was für verschiedene Rollen in Ihrem Unternehmen am besten geeignet ist.

## Unterstützte Browser (Webplattform)

Die Druma-Webplattform — verwendet von Admins, Planern, Unternehmensadministratoren und Kundenservice — funktioniert in allen gängigen modernen Browsern.

| Browser | Mindestversion |
|---------|----------------|
| Google Chrome | 90 oder neuer |
| Mozilla Firefox | 90 oder neuer |
| Microsoft Edge | 90 oder neuer |
| Apple Safari | 14 oder neuer |

<Note>
Google Chrome wird für das beste Erlebnis empfohlen, insbesondere für die Planungstafel und PDF-Vorschaufunktionen.
</Note>

<Warning>
Internet Explorer wird nicht unterstützt. Wenn Ihr Computer nur Internet Explorer hat, installieren Sie bitte Chrome oder Edge, bevor Sie Druma verwenden. Beide sind kostenlos herunterzuladen.
</Warning>

### Erforderliche Browsereinstellungen

Einige Browsereinstellungen müssen aktiviert sein, damit Druma korrekt funktioniert:

- **JavaScript muss aktiviert sein.** Druma setzt für alles JavaScript voraus. Es ist standardmäßig in allen modernen Browsern aktiviert — Sie müssen dies nur prüfen, wenn Sie eine strenge IT-Richtlinie haben, die es möglicherweise deaktiviert hat.
- **Cookies müssen für `druma.io` erlaubt sein.** Druma verwendet Cookies, um Sie angemeldet zu halten. Wenn Sie Cookies jedes Mal löschen, wenn Sie den Browser schließen, müssen Sie sich jedes Mal neu anmelden.
- **Pop-ups von druma.io sollten erlaubt sein.** Druma öffnet PDF-Berichte und Rechnungen in einem neuen Tab oder Pop-up-Fenster. Wenn Ihr Browser Pop-ups von druma.io blockiert, werden diese nicht geöffnet. Erlauben Sie Pop-ups für druma.io in Ihren Browsereinstellungen.

## Empfehlungen zur Bildschirmgröße

Druma funktioniert auf jedem Bildschirm, aber einige Teile der Plattform sind für größere Bildschirme konzipiert.

| Rolle | Empfohlener Bildschirm |
|------|-------------------|
| Planer (Planungstafel) | Desktop oder Laptop, 1920 Pixel breit oder breiter |
| Planer (allgemeine Nutzung) | Desktop oder Laptop, mindestens 1280 Pixel breit |
| Unternehmensadministrator / Admin | Desktop oder Laptop, mindestens 1280 Pixel breit |
| Kundenservice | Desktop oder Laptop, mindestens 1280 Pixel breit |
| Fahrer | Beliebiges modernes Smartphone |

Die Planungstafel — wo Sie Aufträge per Drag-and-Drop auf Fahrzeuge ziehen — ist am einfachsten auf einem breiten Bildschirm zu bedienen. Auf einem 1280-Pixel-Bildschirm ist sie nutzbar, aber enger. Auf einem Telefon oder Tablet wird die Planungstafel nicht empfohlen.

## Fahrer-App (PWA)

Fahrer verwenden die Webplattform nicht. Stattdessen hat jeder Fahrer einen persönlichen Link, der die **Druma-Fahrer-App** im Browser seines Telefons öffnet. Dies ist eine Progressive Web App (PWA) — sie funktioniert wie eine normale App, muss aber nicht aus einem App-Store heruntergeladen werden.

### Android-Telefone

1. Öffnen Sie den persönlichen Link des Fahrers in **Google Chrome** auf dem Android-Telefon.
2. Chrome zeigt ein Banner mit „Zum Startbildschirm hinzufügen" — tippen Sie darauf.
3. Das Druma-Fahrer-App-Symbol erscheint auf dem Startbildschirm wie eine reguläre App.
4. Von nun an öffnet der Fahrer die App durch Tippen auf dieses Symbol.

### iPhone und iPad (iOS)

1. Öffnen Sie den persönlichen Link des Fahrers in **Safari** auf dem iPhone (es muss Safari sein — Chrome auf iOS unterstützt keine PWA-Installation).
2. Tippen Sie auf die Schaltfläche **Teilen** (das Quadrat mit einem nach oben zeigenden Pfeil).
3. Scrollen Sie nach unten und tippen Sie auf **Zum Home-Bildschirm**.
4. Tippen Sie oben rechts auf **Hinzufügen**.
5. Das Druma-Fahrer-App-Symbol erscheint auf dem Startbildschirm.

<Note>
Auf dem iPhone muss die Fahrer-App in Safari geöffnet werden, um sie zum Startbildschirm hinzuzufügen. Wenn der Fahrer normalerweise Chrome auf seinem iPhone verwendet, sollte er den Link für die Ersteinrichtung in Safari kopieren und einfügen.
</Note>

### Was die Fahrer-App erfordert

- Beliebiges Smartphone mit Android 8 oder neuer oder iOS 14 oder neuer
- Eine funktionierende Internetverbindung (mobile Daten oder WLAN)
- Den persönlichen Fahrerlink, der vom Disponenten geteilt wurde

## Internetverbindung

Druma erfordert jederzeit eine aktive Internetverbindung. **Es gibt keinen Offline-Modus.**

- Für die Webplattform reicht eine Standard-Breitband- oder 4G-Verbindung aus.
- Für die Fahrer-App wird eine 3G- oder bessere mobile Datenverbindung benötigt, um Aufträge zu laden und Updates zu übermitteln.

Wenn ein Fahrer während der Fahrt die Verbindung verliert (z. B. in einem Tunnel oder in einem abgelegenen Gebiet), zeigt die App die zuletzt geladenen Informationen an, kann aber keine Updates synchronisieren, bis die Verbindung wiederhergestellt ist. Updates werden gesendet, sobald die Verbindung zurückkehrt.

## Keine VPN- oder Proxy-Probleme

Druma erfordert kein VPN für den Zugriff. Wenn Ihr Unternehmen ein VPN verwendet, sollte Druma weiterhin darüber funktionieren. Wenn Sie jedoch Ladeprobleme haben, versuchen Sie, das VPN vorübergehend zu deaktivieren, um zu sehen, ob das das Problem behebt.
