## CSV-Vorlage herunterladen

Gehen Sie zu **Fuhrpark → Fahrer** und klicken Sie auf **Importieren → Vorlage herunterladen**.

---

## Erforderliche Spalten

| Spalte | Beschreibung |
|---|---|
| `full_name` | Vollständiger Name des Fahrers (Pflicht) |
| `phone_number` | Mobiltelefonnummer |
| `licence_number` | Führerscheinnummer |
| `licence_expiry` | Führerschein-Ablaufdatum (YYYY-MM-DD) |
| `cpc_expiry` | CPC-Ablaufdatum (YYYY-MM-DD, optional) |
| `adr_expiry` | ADR-Ablaufdatum (YYYY-MM-DD, optional) |

---

## Nach dem Import

Für jeden importierten Fahrer erstellt Druma automatisch einen PWA-Link. Sie können die Links über das Fahrerprofil abrufen und per WhatsApp oder SMS weiterleiten.
