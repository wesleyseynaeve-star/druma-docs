## Übersicht

Der Tab Unfälle in Fuhrpark ist ein strukturiertes Protokoll aller Unfälle, Schadenereignisse oder Sicherheitsvorfälle mit Ihren Fahrzeugen.

**Zugriff:**
- Pro Fahrzeug: Fuhrpark → LKWs → Fahrzeug öffnen → Tab **Unfälle**
- Fuhrparkweit: Fuhrpark → **Unfälle**

---

## Unfall erfassen

<Steps>
  <Step title="Unfallansicht öffnen">
    Fahrzeug öffnen und auf Tab **Unfälle** klicken, oder zu **Fuhrpark → Unfälle** navigieren.
  </Step>
  <Step title="Neuen Unfall anklicken">
    Auf **+ Neuer Unfall** klicken. Das Formular öffnet sich.
  </Step>
  <Step title="Details ausfüllen">
    Formularfelder ausfüllen (siehe Tabelle unten).
  </Step>
  <Step title="Dokumente und Fotos anhängen">
    Schadensfotos, Polizeibericht-PDF, Versicherungskorrespondenz oder Fahrererklärung anhängen.
  </Step>
  <Step title="Speichern">
    Auf **Speichern** klicken.
  </Step>
</Steps>

### Formularfelder

| Feld | Beschreibung |
|---|---|
| **Datum und Uhrzeit** | Wann der Unfall passierte |
| **Ort** | Adresse oder GPS-Koordinaten |
| **Fahrer** | Welcher Fahrer das Fahrzeug fuhr |
| **Unfallart** | Kollision / Schaden / Beinahe-Unfall / Diebstahl / Vandalismus / Sonstiges |
| **Beschreibung** | Freitextbericht |
| **Drittpartei beteiligt** | Aktivieren wenn ein anderes Fahrzeug beteiligt war |
| **Unfallberichtsnummer** | Referenz wenn ein Polizeibericht erstattet wurde |
| **Verletzte** | Aktivieren und beschreiben wenn jemand verletzt wurde |

---

## Versicherungsschaden-Verfolgung

| Feld | Beschreibung |
|---|---|
| **Schadensreferenznummer** | Referenz Ihres Versicherers |
| **Schadenstatus** | Offen / In Prüfung / Reguliert / Abgelehnt |
| **Regulierungsbetrag (€)** | Bei Regulierung eingeben |
