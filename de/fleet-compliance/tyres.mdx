## Übersicht

Die Seite Fuhrpark → Reifen verfolgt Reifenmontage, Zustand und Austauschhistorie für jedes Fahrzeug. Sie gibt Fuhrparkmanagern Sichtbarkeit über reifenbezogene Kosten und Verschleißzyklen, ohne auf Papieraufzeichnungen angewiesen zu sein.

**Zugriff:** Fuhrpark → **Reifen** — eine eigenständige Fuhrpark-Hauptseite, kein Tab in der Fahrzeugdetailansicht.

## Reifeneinträge

Reifenpositionen werden frei pro Fahrzeug hinzugefügt — es gibt kein festes Positionsschema und keine Obergrenze, wie viele Positionen ein Fahrzeug haben kann; fügen Sie genau die Positionen hinzu, die das Fahrzeug tatsächlich hat. Jeder Reifeneintrag umfasst:

| Feld | Beschreibung |
|---|---|
| **Position** | Achse und Seite (z. B. Vorne Links, Antriebsachse Rechts Innen) |
| **Marke und Modell** | Reifenhersteller und Produktname |
| **Größe** | Reifengrößenbezeichnung (z. B. 315/70 R22.5) |
| **Seriennummer** | DOT-Code des Reifens zur Rückverfolgbarkeit |
| **Montagedatum** | Wann dieser Reifen eingebaut wurde |
| **Profiltiefe (mm)** | Aktuell gemessene Profiltiefe |
| **Zustand** | Gut / Beobachten / Ersetzen |
| **Notizen** | Freitextnotizen (Schäden, Reifenpannen-Historie usw.) |

---

## Einen Reifeneintrag hinzufügen

<Steps>
  <Step title="Fuhrpark → Reifen öffnen">
    Gehen Sie zur eigenständigen Seite **Fuhrpark → Reifen**.
  </Step>
  <Step title="Das Fahrzeug wählen">
    Wählen Sie das Fahrzeug, für das Sie einen Reifeneintrag hinzufügen möchten.
  </Step>
  <Step title="Auf Reifen hinzufügen klicken">
    Klicken Sie auf **+ Reifen hinzufügen**. Ein Formular öffnet sich.
  </Step>
  <Step title="Details eingeben">
    Wählen Sie die Position aus der Liste und füllen Sie Marke, Größe, Seriennummer, Montagedatum, Profiltiefe und Zustand aus.
  </Step>
  <Step title="Speichern">
    Klicken Sie auf **Speichern**. Der Reifeneintrag erscheint im Positionsraster.
  </Step>
</Steps>

## Reifenzustands-Warnungen

Druma überwacht den Reifenzustand und warnt Fuhrparkmanager automatisch:

| Zustand | Was ihn auslöst | Warnung |
|---|---|---|
| **Beobachten** | Zustand auf Beobachten gesetzt, oder Profiltiefe unter 4 mm | Gelber Indikator auf der Fahrzeugkachel in der Fuhrparkliste |
| **Ersetzen** | Zustand auf Ersetzen gesetzt, oder Profiltiefe unter 2 mm | Roter Indikator auf der Fahrzeugkachel; E-Mail-Warnung an Fuhrparkmanager und Admin |

<Warning>
Eine Profiltiefe unter **2 mm** löst automatisch das Kennzeichen **Ersetzen** aus; unter **4 mm** löst **Beobachten** aus. Disponieren Sie kein Fahrzeug mit als Ersetzen markierten Reifen.
</Warning>

## Reifenzustand aktualisieren

So aktualisieren Sie den Zustand eines Reifens nach einer Inspektion:

1. Öffnen Sie **Fuhrpark → Reifen** und wählen Sie das Fahrzeug
2. Klicken Sie auf **Bearbeiten** in der Reifenzeile
3. Aktualisieren Sie das Feld **Profiltiefe** und/oder **Zustand**
4. Speichern

Dies löscht die Warnung, sobald der Reifen ersetzt oder der Zustand korrigiert wurde.

---

## Einen Reifenwechsel erfassen

Wird ein Reifen physisch ersetzt:

1. Bearbeiten Sie den bestehenden Reifeneintrag für diese Position
2. Setzen Sie **Zustand** auf **Ersetzt**
3. Geben Sie die **Ersatzkosten (€)** ein
4. Klicken Sie auf **Speichern**

Alternativ klicken Sie auf **+ Reifen hinzufügen** an derselben Position, um den neuen Reifen hinzuzufügen (Druma markiert den alten Eintrag als historisch).

Die Ersatzkosten fließen in den Wartungskostenbericht des Fahrzeugs ein — für eine Gesamtbetriebskostenansicht neben Kraftstoff- und Werkstattkosten.

---

## Reifenhistorie

Druma führt eine Historie aller Reifeneinträge pro Position, einschließlich ersetzter Reifen. So sehen Sie die Historie für eine bestimmte Position:

1. Öffnen Sie **Fuhrpark → Reifen** und wählen Sie das Fahrzeug
2. Klicken Sie auf die Position im Raster
3. Das Historienpanel zeigt alle jemals an dieser Position montierten Reifen mit Montagedatum, Zustandsstatus und Ersatzkosten
