Smart Import iti permite sa incarci documente existente (PDF-uri, imagini, Excel, CSV) si AI-ul Druma (Gemini 2.0 Flash) extrage automat datele structurate pentru inregistrarea in masa.

## Ce date poate extrage

- **Companie** - date de inregistrare, TVA, adresa, contact
- **Flotă** - numere de inmatriculare, tipuri de vehicule, date de expirare documente
- **Soferi** - nume, numere de telefon, detalii permis, date CPC/ADR
- **Clienti** - denumiri companii, TVA, adrese, contacte
- **Asigurari** - numere de polita, date de expirare, asigurator

## Fluxul de lucru

<Steps>
  <Step title="Mergi la Tools -> Import Inteligent">In meniul principal, mergi la Instrumente -> Import Inteligent.</Step>
  <Step title="Incarca documentele">Trage si plaseaza fisierele PDF, imagini, Excel sau CSV. Dimensiune maxima 10 MB per fisier.</Step>
  <Step title="Asteapta extractia AI">Gemini analizeaza documentele si extrage campurile cu un scor de incredere per camp.</Step>
  <Step title="Revizuieste dashboard-ul">Examineaza datele extrase pe categorii. Editeaza inline campurile incorecte.</Step>
  <Step title="Elimina duplicatele">Druma verifica automat duplicatele fata de datele existente ale companiei.</Step>
  <Step title="Importa in lot">Selecteaza inregistrarile de importat si apasa Importa.</Step>
</Steps>

<Note>
Extractia AI nu este 100% precisa. Revizuieste intotdeauna datele extrase inainte de a importa.
</Note>

## Formate acceptate

PDF, JPG, PNG, HEIC, XLSX, CSV.
