| Rol | Descriere |
|-----|-----------|
| Admin | Acces complet inclusiv facturare |
| Company Admin | Acces complet fara facturare |
| Planner | Comenzi, atribuire, facturare, rapoarte |
| Dispatcher | Monitorizare (doar citire) |
| Fleet Manager | Gestionare flota completa |
| Customer Service | Clienti, comenzi, facturi, invitare utilizatori |
| Driver | Numai aplicatia mobila |
| Client | Numai portalul clientului |

## Admin
Acces nerestrictionat la fiecare functie Druma. Rolul pentru proprietarul afacerii.

## Company Admin
Identic cu Admin fara accesul la facturare. Ideal pentru directori operatiunii.

## Planner
Creeaza comenzi, atribuie soferi si camioane, genereaza facturi, acceseaza rapoarte.

## Dispatcher
Rol de vizualizare. Vede tabloul de bord, harta, statusurile comenzilor. Nu poate crea sau edita nimic.

## Fleet Manager
Acces complet la flota: vehicule, remorci, soferi, documente, combustibil, cabotaj, ore de conducere.

## Customer Service
Similar cu Planner cu accent pe clienti. Poate invita utilizatori noi.

## Driver
Acceseaza numai aplicatia mobila PWA: vede comenzile, actualizeaza statusul, semneaza eCMR, incarca documente.

## Client
Acceseaza numai portalul clientului: urmareste transporturile, descarca documente.
