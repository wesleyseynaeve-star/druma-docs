Mergi la **Setari -> Integrarsi** (Admin sau Company Admin).

## 1. Furnizor eCMR

Druma emite scrisori de trăsură electronice **intern, folosind furnizorul nativ Druma în mod implicit** — nu este necesar niciun cont extern, cheie API sau configurare. eCMR este disponibil imediat pentru comenzile transfrontaliere, iar Druma sigilează fiecare document finalizat cu o semnătură digitală PAdES.

**TransFollow** rămâne disponibil ca alternativă opțională pentru companiile care îl foloseau înainte de introducerea furnizorului nativ. Companiile noi ar trebui să rămână pe furnizorul nativ.

<Steps>
  <Step title="Deschide cardul furnizorului eCMR">Mergi la **Setări → Integrări** și deschide cardul **Furnizor eCMR**. Furnizorul nativ Druma este selectat implicit.</Step>
  <Step title="(Opțional) Comutare la TransFollow">Pentru a utiliza TransFollow în schimb, selectează-l și lipește cheia API din tabloul de bord al contului tău TransFollow în câmpul **Cheie API**. Înregistrează-te la [transfollow.org](https://transfollow.org) dacă nu ai cont.</Step>
  <Step title="Activează modul simulare (opțional)">Când folosești TransFollow, activează **Modul Simulare** pentru a testa fluxul eCMR fără a emite documente reale cu valabilitate legală. Folosește aceasta în timpul instruirii sau onboardingului șoferilor. Dezactivează-l înainte de a începe activitatea reală.</Step>
  <Step title="Testează conexiunea">Apasă **Testează Conexiunea**. Druma verifică certificatul de semnare nativ — sau, dacă TransFollow este selectat, apelează API-ul TransFollow pentru a confirma că cheia ta este validă.</Step>
  <Step title="Salvează">Apasă **Salvează**. Crearea eCMR este activă pentru comenzile transfrontaliere.</Step>
</Steps>

<Note>
Majoritatea companiilor nu trebuie să schimbe niciodată această setare — furnizorul nativ este implicit recomandat. Consultați [Furnizori eCMR](/ro/ecmr/ecmr-providers) pentru o comparație completă.
</Note>

<Warning>
Documentele eCMR în modul simulare TransFollow **nu sunt valabile din punct de vedere legal**. Asigură-te că modul simulare este dezactivat înainte de a expedia transporturi reale.
</Warning>

## SmartBill

Mergi la Setari -> Integrarsi -> SmartBill, introdu email-ul SmartBill si cheia API, apasa Testeaza.

## SAGA / WinMENTOR

Fara conexiune directa. Mergi la Facturi -> Export SAGA sau Export WinMENTOR pentru fisierul ZIP contabil.

## Carduri de combustibil

Druma accepta exporturi CSV de la DKV, UTA si AS24. Mergi la Flota -> Combustibil -> Import.

## HERE Maps

Configurat la nivel de platforma - nu necesita configurare.

## ANAF e-Transport

Mergi la Setari -> Integrarsi -> ANAF e-Transport si introdu credentialele SPV.
