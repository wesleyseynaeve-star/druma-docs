Mergi la **Setari -> Branding si Formate**. Trei taburi: Branding, Format Factura, Format Comanda.

## Tabul Branding

- **Logo** - PNG, JPG sau SVG, minim 300 px latime. Apare pe facturile PDF si email-uri.
- **Culoarea principala** - pentru facturile PDF
- **Culoarea accent** - pentru elementele secundare

<Note>
Modificarile se aplica numai documentelor noi.
</Note>

## Tabul Format Factura

Token-uri: `{PREFIX}`, `{YYYY}`, `{YY}`, `{MM}`, `{NNN}`, `{NNNN}`

**Format implicit:** `DR-{YYYY}-{NNN}` produce `DR-2026-001`

Note de credit: format implicit `CN-{YYYY}-{NNN}`

## Tabul Format Comanda

**Format implicit:** `DR-{YYYY}-{NNN}`

<Warning>
Schimbarea formatului la mijlocul anului nu renumeroteaza facturile existente.
</Warning>
