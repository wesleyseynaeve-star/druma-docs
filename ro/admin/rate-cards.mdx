Un tarif este o colectie de preturi pe rute. Cand creezi o comanda, Druma sugereaza pretul corect automat.

## Crearea unui tarif

Mergi la **Setari -> Tarife** si apasa **Tarif Nou** (Nume, Moneda implicita, Intrari).

## Intrari tarif

| Camp | Descriere |
|------|-----------|
| Tara de origine | Tara de preluare |
| Tara de destinatie | Tara de livrare |
| Tip pret | Suma forfetara sau pret per km |
| Suma | Pretul in moneda selectata |
| Tip remorca | Optional |
| ADR supliment | Supliment pentru marfuri periculoase |

## Atribuirea tarifelor la clienti

Mergi la **Clienti** - deschide clientul - tabul **Termeni comerciali** - selecteaza tariful - Salveaza.
