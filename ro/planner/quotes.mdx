Mergi la **Comenzi -> Oferte**. Apasa **Oferta Noua**: client, ruta, marfa, pret, data valabilitatii, note.

## Statusuri

| Status | Descriere |
|--------|-----------|
| Ciorna | Creata, netrimisa |
| Trimisa | In asteptarea raspunsului |
| Acceptata | Clientul a acceptat |
| Respinsa | Clientul a respins |
| Expirata | Data valabilitatii a trecut |
| Convertita | Transformata intr-o comanda |

## Conversia intr-o comanda

Apasa **Converteste in Comanda**. Druma creeaza o comanda pre-completata in status Ciorna.
