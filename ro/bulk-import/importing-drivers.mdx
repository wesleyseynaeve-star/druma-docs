---
title: "Importul soferilor"
description: "Incarca in masa soferii in Druma folosind un template CSV."
icon: "file-import"
---

<Steps>
  <Step title="Deschide pagina de import">Mergi la **Camioane -> Soferi -> Import** si apasa **Descarca Template**.</Step>
  <Step title="Completeaza CSV-ul">Un sofer per rand: nume complet, telefon (format international), email, numar permis, categorie permis (CE), data expirare permis, data expirare CPC, card tahograf, data expirare ADR, tara, activ (true/false).</Step>
  <Step title="Incarca si revizuieste">Verifica randurile rosii (de obicei numar telefon lipsa sau format data invalid - AAAA-LL-ZZ).</Step>
  <Step title="Confirma importul">Apasa **Importa [X] soferi**.</Step>
</Steps>

Numarul de telefon este critic - soferii se autentifica cu acesta in PWA. Trebuie sa fie in format international (+40...).

Dupa import, distribuie linkul PWA fiecarui sofer: **Camioane -> Soferi -> [sofer] -> Copiaza Link PWA**.
