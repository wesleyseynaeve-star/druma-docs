---
title: "Integrarea SmartBill"
description: "Trimite facturile Druma direct in SmartBill cu un singur clic."
icon: "file-invoice"
---

<Steps>
  <Step title="Obtine cheia API">Autentifica-te la smartbill.ro, mergi la Setari -> Integrari API si genereaza o cheie noua.</Step>
  <Step title="Introdu in Druma">Mergi la **Setari -> Integrarsi -> SmartBill**, introdu email-ul si cheia API.</Step>
  <Step title="Testeaza">Apasa **Testeaza** si verifica ca compania ta este gasita.</Step>
</Steps>

Pe orice factura generata, apasa **Trimite in SmartBill**. Druma trimite datele facturii la SmartBill.

Sincronizarea este unidirectionala (Druma -> SmartBill). Modificarile din SmartBill nu se reflecta in Druma.
