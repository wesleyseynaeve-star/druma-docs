Druma ofera un API REST public. Genereaza o cheie API in **Setari -> API -> Genereaza Cheie API**.

Include cheia in fiecare cerere: `Authorization: Bearer CHEIA_TA_API`

Limita de rata: 100 cereri per minut.

Documentatia completa: **druma.io/docs/api**

## Endpoint-uri principale

- `GET /v1/orders` - Lista comenzilor
- `POST /v1/orders` - Creeaza o comanda
- `GET /v1/orders/{id}` - Detaliile comenzii
- `PUT /v1/orders/{id}/status` - Actualizeaza statusul
- `GET /v1/clients` - Lista clientilor
- `GET /v1/fleet` - Lista vehiculelor
