## Platforma web

| Browser | Versiune minima |
|---------|----------------|
| Google Chrome | 110+ (recomandat) |
| Microsoft Edge | 110+ |
| Mozilla Firefox | 110+ |
| Safari macOS | 16+ |

Tabla de planificare necesita minim **1280 px latime** pentru drag-and-drop. Celelalte pagini functioneaza de la 768 px.

## Aplicatia mobila

- **Android:** Android 8.0+, Google Chrome, Google Play
- **iOS:** iOS 16.4+, Safari (necesar pentru instalarea PWA)

Functioneaza cu semnal 3G sau mai bun.

## Portalul clientului

Functioneaza pe orice browser modern. Nu necesita instalare.
