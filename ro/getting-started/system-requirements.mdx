## Platforma web

Nimic de instalat - un browser modern si o conexiune stabila la internet sunt suficiente.

| Browser | Versiune minima |
|---------|----------------|
| Google Chrome | 90+ (recomandat) |
| Microsoft Edge | 90+ |
| Mozilla Firefox | 90+ |
| Apple Safari | 14+ |

Internet Explorer nu este suportat. JavaScript si cookie-urile pentru `druma.io` trebuie permise, iar pop-up-urile trebuie acceptate (pentru rapoarte PDF si facturi).

Tabla de planificare necesita minim **1920 px** latime pentru o experienta confortabila (utilizabila de la 1280 px); restul platformei functioneaza de la 1280 px. Pe telefon sau tableta, tabla de planificare nu este recomandata.

## Aplicatia soferului

Soferii nu folosesc platforma web - deschid **aplicatia Druma pentru soferi** (aceeasi adresa pentru toti soferii companiei) si se autentifica cu **numar de telefon + PIN** configurat de dispecer (vezi [Instalarea aplicatiei soferului](/ro/driver/installing-the-app)).

- **Android:** aplicatie nativa din Google Play, Android 8.0+, Google Chrome pentru instalare initiala
- **iOS:** PWA instalata din Safari (obligatoriu Safari - Chrome pe iOS nu suporta instalarea PWA), iOS 14+

Necesita conexiune activa (3G/4G/Wi-Fi) - **nu exista mod offline**. Daca soferul pierde semnalul (tunel, zona izolata), aplicatia arata ultimele date incarcate si sincronizeaza actualizarile cand conexiunea revine.
