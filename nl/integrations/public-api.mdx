De Druma Openbare API laat u aangepaste integraties bouwen tussen Druma en uw eigen systemen — een ERP, een klantenzelfbedieningsportaal, een business intelligence-tool of interne tooling.

<Warning>
De Openbare API is een functie die standaard niet is ingeschakeld. Neem contact op met **support@druma.io** om toegang aan te vragen voor uw account.
</Warning>

## API-sleutelbeheer

Zodra de Openbare API is ingeschakeld voor uw account, ga naar **Instellingen → API & Webhooks** om uw API-sleutel te genereren en te beheren.

## Verificatie

Druma gebruikt Bearer Token-verificatie. Voeg uw API-sleutel toe aan de Authorization-header van elke request:

```
Authorization: Bearer uw_api_sleutel
```

## Beschikbare endpoints

- `GET /orders` — lijst van orders met filtering op status, datum, klant
- `GET /orders/{id}` — orderdetails inclusief stops en statusgeschiedenis
- `POST /orders` — nieuwe order aanmaken
- `PATCH /orders/{id}/status` — orderstatus bijwerken
- `GET /clients` — klantlijst
- `GET /fleet` — voertuig- en chauffeurslijst

## Webhooks

Configureer webhook-endpoints om realtime meldingen te ontvangen voor:
- Order aangemaakt
- Status bijgewerkt (voor elke statusovergang)
- Afgeleverd
- Factuur aangemaakt
