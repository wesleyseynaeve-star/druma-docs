## Преглед

Импортирайте регистъра на подизпълнителите наведнъж.

## Изтегляне на шаблона

Отидете в **Спедиция → Подизпълнители → Масов внос → Изтеглете шаблона**.

## Колони

| Колона | Задължително | Описание |
|--------|-------------|----------|
| company_name | Да | Наименование на превозвача |
| vat_number | Да | ДДС номер (за дедупликация) |
| country | Не | ISO 2-буквен код |
| contact_name | Не | Контактно лице |
| contact_email | Не | Имейл |
| contact_phone | Не | Телефон |
| payment_terms_days | Не | Срок на плащане |

## Дедупликация

Подизпълнители с вече съществуващ ДДС номер се пропускат.
