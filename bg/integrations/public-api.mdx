## Преглед

Публичният API на Druma позволява на трети системи да интегрират с Druma програматично.

## Автентикация

Всички API заявки изискват JWT токен в заглавката:
```
Authorization: Bearer <token>
```

Получете токен от **Настройки → API → Генерирай токен**.

## Основни крайни точки

| Метод | Маршрут | Описание |
|-------|---------|----------|
| GET | /orders | Списък с поръчки |
| POST | /orders | Създаване на поръчка |
| GET | /orders/{id} | Детайли на поръчка |
| PATCH | /orders/{id} | Актуализиране на поръчка |
| GET | /fleet | Списък на превозни средства |
| GET | /drivers | Списък на шофьори |

## Rate Limiting

100 заявки в минута на компания.

## Уеб хукове

Конфигурирайте уеб хук URL за получаване на известия при:
- Промяна на статус на поръчка
- Доставка потвърдена
- Фактура изпратена
