## Преглед

Отчетът за доходност по маршрути показва колко печелите по всяка двойка произход-дестинация.

## Гранулярност

Превключвайте между:
- **Страна** — показва страни (напр. РО → ДЕ)
- **Град** — показва градове (напр. Букурещ → Мюнхен)

## Колони

| Колона | Описание |
|--------|----------|
| Маршрут | Произход → Дестинация |
| Поръчки | Брой поръчки за периода |
| Приход | Общ приход |
| Разходи | Гориво, такси, шофьори |
| Марж | Приход минус разходи |
| Марж % | Процент марж |
| CO2 (кг) | Общи емисии |
| Навреме % | Процент доставени навреме |

## Сортиране

Кликнете колона за сортиране. Сортирайте по марж, за да намерите най-доходните маршрути.

## Износ

Кликнете **Експорт CSV** за детайлен анализ в Excel.

<Note>
Отчетът е ограничен до 500 записа. При по-голям обем стесните периода.
</Note>
