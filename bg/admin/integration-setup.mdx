## Преглед

Всички интеграции се конфигурират в **Настройки → Интеграции**.

| Интеграция | Цел |
|---|---|
| SmartBill | Изпращане на фактури |
| SAGA / WinMENTOR | Счетоводни файлове |
| ANAF e-Factura | Подаване на фактури към НАП |
| ANAF e-Transport | Деклариране на стоки с висока стойност |
| TransFollow | eCMR документи |
| Peppol | Белгийско B2B е-фактуриране |
| Банкова кореспонденция | Автоматично съпоставяне на плащания |
| HERE Routing | Маршрути, ETA, пътни такси |

## SmartBill

<Steps>
  <Step title="Отидете в Настройки → Интеграции → SmartBill">
    Намерете секцията SmartBill.
  </Step>
  <Step title="Въведете API ключа">
    Копирайте от таблото на SmartBill.
  </Step>
  <Step title="Тествайте">
    Кликнете **Тест**.
  </Step>
</Steps>

## 1. Доставчик на eCMR

Druma издава електронни товарителници **вътрешно, използвайки вградения доставчик на Druma по подразбиране** — не се изисква външен акаунт, API ключ или настройка. eCMR е достъпен от само себе си за международни поръчки, а Druma запечатва всеки завършен документ с PAdES цифров подпис.

**TransFollow** остава наличен като избираема алтернатива за компании, вече използващи го преди въвеждането на вградения доставчик. Новите компании трябва да останат на вградения доставчик.

<Steps>
  <Step title="Отворете картата за доставчик на eCMR">
    Отидете в **Настройки → Интеграции** и отворете картата **Доставчик на eCMR**. Вграденият доставчик на Druma е избран по подразбиране.
  </Step>
  <Step title="(По избор) Преминете към TransFollow">
    За да използвате TransFollow вместо това, изберете го и поставете API ключа от таблото на вашия акаунт в TransFollow в полето **API ключ**. Регистрирайте се на [transfollow.org](https://transfollow.org), ако нямате акаунт.
  </Step>
  <Step title="Активирайте тестов режим (по избор)">
    При използване на TransFollow включете **Тестов режим**, за да тествате работния процес на eCMR без издаване на реални, законово регистрирани документи. Използвайте го при обучение на шофьори или въвеждане в работа. Изключете го преди да влезете в реална работа.
  </Step>
  <Step title="Тествайте връзката">
    Кликнете **Тест на връзката**. Druma проверява вградения сертификат за подписване — или, ако е избран TransFollow, извиква API на TransFollow, за да потвърди, че ключът е валиден.
  </Step>
  <Step title="Запазете">
    Кликнете **Запази**. Създаването на eCMR е активно за международни поръчки.
  </Step>
</Steps>

<Note>
Повечето компании никога не се налага да променят това — вграденият доставчик е препоръчаната настройка по подразбиране. Вижте [Доставчици на eCMR](/en/ecmr/ecmr-providers) за пълно сравнение.
</Note>

<Warning>
eCMR документите в тестов режим на TransFollow **не са правно валидни**. Уверете се, че тестовият режим е изключен преди изпращане на реални пратки.
</Warning>
