## Статусите на поръчката

| Статус | Значение |
|--------|---------|
| **Чакаща** | Поръчката е създадена, но не е назначено превозно средство |
| **Назначена** | Назначено е превозно средство и шофьор |
| **При товарене** | Шофьорът е пристигнал на адреса за товарене |
| **Товари се** | Шофьорът е потвърдил началото на товаренето |
| **В движение** | Товарът е натоварен, шофьорът е тръгнал |
| **При доставка** | Шофьорът е пристигнал на адреса за доставка |
| **Разтоварва се** | Шофьорът е потвърдил началото на разтоварването |
| **Доставена** | Доставката е завършена |
| **Отменена** | Поръчката е анулирана |

## Какво задейства всяка стъпка

### Чакаща → Назначена
Планиращият назначава превозно средство и шофьор. Шофьорът получава push известие.

### Назначена → При товарене
Шофьорът докосва **Пристигнах на товарене** в приложението.

### При товарене → Товари се
Шофьорът докосва **Започвам товарене**.

### Товари се → В движение
Шофьорът докосва **Тръгвам**. GPS проследяването се активира.

### В движение → При доставка
Шофьорът докосва **Пристигнах на доставка**.

### При доставка → Разтоварва се
Шофьорът докосва **Започвам разтоварване**.

### Разтоварва се → Доставена
Шофьорът докосва **Готово**. Клиентът е уведомен. Може да се генерира фактура.

## Автоматизации при доставка

При маркиране като Доставена:
- Клиентът получава имейл потвърждение (ако е активирано)
- Може да се генерира автоматична чернова фактура
- eCMR документът е финализиран
